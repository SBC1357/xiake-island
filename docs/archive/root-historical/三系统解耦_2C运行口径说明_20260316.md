## 2. Fail-Fast 行为清单

### 2.1 V2 (medical_kb_system_v2)

| 场景 | 异常类型 | 行为 |
|------|----------|------|
| KNOWLEDGE_ROOT 未配置 | `KnowledgeRootNotConfiguredError` | 启动/运行时失败 |
| 知识根目录不存在 | `KnowledgeRootNotFoundError` | 启动/运行时失败 |
| 知识根目录不可读 | `KnowledgeRootNotReadableError` | 启动/运行时失败 |
| consumer 目录不存在 | `KnowledgeRootNotFoundError` | 启动/运行时失败 |

**Fail-Fast 位置**:
- `engine/config/knowledge_source.py`: 配置验证
- `engine/evidence/external_resolver.py`: 运行时解析

### 2.2 侠客岛 (Xiakedao)

| 场景 | 异常类型 | 行为 |
|------|----------|------|
| XIAGEDAO_CONSUMER_ROOT 未配置 | `ValueError` | 启动时失败 (严格模式) |
| consumer 目录不存在 | `ValueError` | 启动时失败 (严格模式) |
| consumer 目录不可读 | `ValueError` | 启动时失败 (严格模式) |

**Fail-Fast 位置**:
- `src/assembly.py`: `get_asset_bridge()` 严格模式验证

## 3. 显式回退开关

### 3.1 V2 Legacy 模式

**启用方式**:
```powershell
$env:KNOWLEDGE_SOURCE_MODE = "legacy"
$env:V1_BASE_PATH = "D:\汇度编辑部1\写作知识库\medical_kb_system"
```

**行为**:
- 使用 `LegacyResolver` 读取 V1 格式知识
- 临时用于紧急回退或数据迁移验证
- **不推荐长期使用**

### 3.2 侠客岛非严格模式

**启用方式**:
```powershell
$env:XIAGEDAO_STRICT_MODE = "false"
```

**行为**:
- 允许无 consumer 配置启动
- 仅打印警告，不阻止应用启动
- 用于本地开发/测试环境

## 7. 故障排查

| 现象 | 可能原因 | 解决方案 |
|------|----------|----------|
| KnowledgeRootNotConfiguredError | 环境变量未加载 | 执行 `set-default-env.ps1` |
| ValueError: 消费者配置无效 | XIAGEDAO_CONSUMER_ROOT 错误 | 检查路径是否存在 |
| 空结果 | legacy 模式误开启 | 确认 KNOWLEDGE_SOURCE_MODE=external |
