## 2. 前端入口暴露清单

### 2.1 已完成（可直接使用）

| 入口 | 组件 | 功能 |
|------|------|------|
| 任务历史面板 | `HistoryPanel.tsx` | 查看历史任务、加载参数 |
| 反馈表单 | `FeedbackForm.tsx` | 提交评分、标签、评论 |
| 结果操作 | `ResultActions.tsx` | 复制、导出 Markdown、提交反馈 |
| 基于历史重新执行 | `TriggerPage.tsx` | basedOnTaskId 标记、确认覆盖 |

### 2.2 待暴露（需新增）

| 入口 | 依赖后端 | 建议组件 | 优先级 |
|------|---------|---------|--------|
| 版本查询 UI | A-7 版本 API | `TaskVersionList.tsx` | P2 |
| 回退确认 UI | A-7 回退 API | 集成到 HistoryPanel | P2 |
| 规划参数输入 | B-1 PlanningService | `PlanningForm.tsx` | P1 |
| 写作生成触发 | B-2 WritingService | 集成到 TriggerPage | P1 |

### 2.3 不需要独立入口

| 能力 | 原因 |
|------|------|
| 质量门禁 | 内部流程，写作生成后自动调用 |
| 交付输出 | 自动生成，无需用户操作 |
| 审计事件 | 内部记录，无编辑价值 |

## 3. 编辑部工作流分析

### 3.1 当前可行流程

```
查看历史 → 加载参数 → 修改表单 → 执行 → 查看结果 → 提交反馈
```

### 3.2 缺失环节

1. **版本对比**：无法查看同一输入的历史版本
2. **回退确认**：无法在 UI 确认回退操作
3. **规划工作流**：无法使用选题/规划能力
