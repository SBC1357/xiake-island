### 1.1 能力分析

| QualityOrchestrator | 内部流程 | 无需入口 |
| MarkdownWriter | 自动生成 | 无需入口 |

### 1.2 集成方案

根据"先接核心流程"原则，B线能力作为后台服务集成：

1. **规划能力**：在 Opinion/SemanticReview 执行前调用 PlanningService
2. **写作生成**：在提交前调用 WritingService 编译 prompt
3. **质量门禁**：在生成后自动运行 QualityOrchestrator
4. **交付输出**：结果自动导出 Markdown

### 1.3 前端已有入口

- 观点生成表单：OpinionConfigForm
- 语义审核表单：SemanticReviewConfigForm
- 文章工作流表单：ArticleWorkflowConfigForm
- 执行触发：TriggerPage
