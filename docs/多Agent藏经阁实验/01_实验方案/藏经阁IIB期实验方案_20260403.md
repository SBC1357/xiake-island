## 一、这一期只回答什么

`IIB` 第一轮只回答 4 个问题：

1. 当前 `动作 + 构件 + 规则 + 拓扑 + 人格` 的骨架，面对新的混合型样本时，是否仍只需要补边界而不需要翻骨架。
2. `条件性对象表` 里的争议条目，哪些能稳定留下，哪些仍只是弱证据。
3. 当前体系能否稳定处理 3 类高风险混合稿：
   - 人物 + 行业/问题
   - 公司史/品类史 + 产业分析
   - 无单一主角的现场报道型问题稿
4. 一个样本进来后，是否能给出 `不超过 5 种` 可执行写法路线，而不是只做事后解释。

## 二、这一期不做什么

这一期先不做 3 件事：

1. 不再争“人格”要不要改回别的名字
2. 不以扩大样本量为目的再开一轮候选筛选
3. 不在第一轮里引入新的底层第五层

## 四、执行方法

三个中心都执行同一套流程：

1. 先独立跑完 `5` 篇公共主样本。
2. 每篇样本都必须回答下面 `7` 个固定问题：
   - 主人格是什么
   - 主拓扑是什么
   - 触发了哪些条件性对象 / 稿型子型
   - 哪些旧规则失效或需要降级
   - 这篇样本最可能暴露哪条边界
   - 如果要反向调用，应给出哪 `1-5` 条写法路线
   - 最终你会把它路由到哪一条主路线
3. 主样本完成后，先看三中心是否满足补充样本触发条件。
4. 只有在补充样本触发条件成立时，才追加跑 `刘家琨` 和 `蔡磊`。
5. 三中心独立完成后，再按方案里的互读触发条件进入互读。

## 五、补充样本触发条件

满足任一条件，即追加两篇补充样本：

1. 跑完 `5` 篇公共主样本后，仍有 `2` 条及以上结构争议无法收束
2. 或有 `2` 篇及以上样本在 `人格 / 拓扑 / 条件性对象` 三项上持续严重分裂

如果上述条件都不成立，`刘家琨` 与 `蔡磊` 留作备选，不进入本轮主结论。
