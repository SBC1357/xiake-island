## 一、总原则

1. 这张表只服务于 `IIC` 盲评，不回写 `IIA/IIB`。
2. 主卷使用 `100` 分制；附加仿写对照题也使用 `100` 分制，但单独记分。
3. `IIC` 的最终阶段判断，仍以 [藏经阁IIC期实验方案_20260404.md](藏经阁IIC期实验方案_20260404.md) 的通过门槛为主；本表负责把“写得好不好”压成可比较分数。
4. 任何数值打分，都不能覆盖硬失败项。先判硬失败，再给分。

## 二、硬失败项

出现任一条，该卷最高只能给 `59` 分：

1. 编造素材之外的事实
2. 明显跑错稿型
3. 主承重信息写丢，导致文章核心判断站不住
4. 把噪音或次要信息误当主线
5. `D类` 或附加题明显不执行限制条件
6. 直接暴露原稿身份线索

说明：

- `59` 不是建议分，而是上限。
- 如果同时出现多条硬失败，可以直接判 `0-40`。

## 三、分数档位解释

### `85-100`

强。结构、判断、取舍都成立，足以进入主结论。

### `75-84`

合格。整体成立，有局部缺口，但不影响进入主判断。

### `60-74`

勉强成立。能看出系统在工作，但问题已经明显，需要在汇总中点名。

### `<60`

失败。不是局部波动，而是本卷目标没有完成。

## 四、单卷评分细则

### A类：校准卷

总分 `100`

1. 骨架调用：`30`
   - 是否能正确调用当前四层骨架，而不是写成无意识流水账
2. 信息取舍：`20`
   - 是否能抓住主承重信息，丢掉明显次要信息
3. 结构清晰：`20`
   - 开头、推进、收束是否清楚
4. 可读性：`15`
   - 文字是否顺、是否让人愿意读完
5. 噪音处理：`15`
   - 是否把噪音留在边上，而不是吃成主线

### B类：半结构卷

总分 `100`

1. 自主立意：`20`
   - 在提示变少后，能否自己抓到主问题
2. 结构组织：`25`
   - 是否能把材料组织成成立的文章
3. 主次分层：`20`
   - 是否知道哪些信息该重、哪些该轻
4. 细节买权：`15`
   - 是否靠细节把判断买下来
5. 可读性：`10`
6. 噪音处理：`10`

### C类：实战卷

总分 `100`

1. 找题眼：`25`
   - 能否在更脏的材料池里找到真正的主问题
2. 结构推进：`20`
   - 不是材料堆砌，而是有推进关系
3. 主次判断：`20`
   - 能否压住枝节，不被素材拖跑
4. 材料驯服：`20`
   - 混排引语、背景、重复信息是否被驯服成文章
5. 文字完成度：`15`

### D类：压限卷

总分 `100`

1. 结构家族逼近：`25`
   - 是否逼近原稿的结构家族，而不是只对上题材
2. `人格 / 温度 / 证据密度` 执行：`20`
3. 主拓扑命中：`20`
4. 可读性：`15`
5. 禁用项与噪音处理：`10`
6. 身份不泄漏：`10`

### 附加题：仿写对照卷

总分 `100`

1. 人物核心轴命中：`30`
   - 人物驱动力、伤口或执念是否被重新立住
2. 人物-问题回钉：`25`
   - 行业、产品、解释段落是否反复回到人物本身
3. 结构家族逼近：`20`
   - 是否像同一家族，而不是普通行业人物稿
4. 材料买权：`15`
   - 是否用细节、场景、数据买下判断
5. 文气完成度：`10`

## 五、主卷总分计算

`IIC` 主卷总分不做简单平均，使用加权：

1. `A类`：`15%`
2. `B类`：`20%`
3. `C类`：`30%`
4. `D类`：`35%`

公式：

`IIC主卷总分 = A×0.15 + B×0.20 + C×0.30 + D×0.35`

附加仿写对照题不并入主卷总分，单列为“旁证分”。

## 六、建议通过线

以下是总协调建议线，用于最后收口时辅助判断：

1. `IIC主卷总分 >= 78`
2. `A类 >= 70`
3. `D类 >= 75`
4. 不得出现硬失败
5. 如果附加仿写对照题大多数执行中心 `<70`，不直接判 `IIC` 失败，但必须在终验中单列：`控制精度不足`

## 七、盲评记录格式

总协调盲评时，建议每卷至少记录：

1. 本卷分数
2. 是否触发硬失败
3. 最强一条优点
4. 最关键一条失分点
5. 是否进入主结论

最小记录模板：

```md
- 卷别：
- 分数：
- 是否硬失败：
- 最强优点：
- 关键失分点：
- 结论：进入主结论 / 仅作旁证 / 失败
```

## 八、使用边界

1. 这张表只是一把统一尺子，不替代总协调的文字判断。
2. 当文字判断与数值分数冲突时，必须把冲突原因写出来，不能只报一个分。
3. 如果后续 `IIC` 扩一轮，这张表可以沿用；若题型结构改变，再单独修订版本。
