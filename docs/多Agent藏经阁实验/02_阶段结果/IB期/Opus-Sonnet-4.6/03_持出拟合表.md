# IB期持出拟合表 — Opus/Sonnet 4.6

## 本文件作用

对每篇实际读取的原文，用候选四层模型(动作/构件/规则/拓扑)做"持出(extract)"，然后检查模型能否"拟合(fit)"——即：用四层表达是否能完整、无遗漏地还原这篇文章的结构和手法。

---

## 拟合一：半佛仙人《钻石崩了？因为不够消费主义》

### 原文结构速写

1. **开篇(section 1)**：读者提问引入 → 反转("恰恰相反朋友") → 再锤("钻石崩了就是因为不够消费主义") → 拦截预判("有人会说培育钻冲击") → 再反转("也不是的朋友")
2. **里子(section 2-3)**：设问引导("那问题来了") → 框架提出("面子+里子") → 先论证里子 → 类比链(包包/手表/茅台/劳力士 vs 钻石) → 降维判断("就干割你啊")
3. **面子(section 4-5)**：框架另一半 → 消费主义本质降维("你消费什么你就是什么") → logo论证 → 钻石尺寸悖论(太小打不了logo→但大了肯定是假的) → 爱情锚定 vs 奢侈品玄学锚定
4. **收束(section 6)**：拦截延伸质疑("翡翠天珠呢？") → 降维回答("人家坑有钱人不坑年轻人结婚钱") → 总判("哪哪都不行") → 锤击收束("能撑到今天已经是奇迹了")

### 四层拟合

| 层 | 提取内容 | 拟合质量 |
|----|----------|----------|
| 动作 | A05(反转开篇) → A08(设问引导) → A06(降维类比)×多次 → A07(重复锤击) → A05(再次反转拦截) | ✅ 完整覆盖，无遗漏动作 |
| 构件 | C07(反共识转折块)×开篇 + C06(跨域重定义块)×里子/面子 + C09(重复锤击块)×收束 | ✅ 全文可切分为3-4个构件单元 |
| 规则 | R06(前3段翻转)✅ R07(关键判断锤3次)✅ R08(降维到菜市场)✅ R10(第二人称"你")✅ | ✅ 四条均验证 |
| 拓扑 | T04(降维论证弧：反转→降维→编号分章→递进→锤击→收束) | ✅ 精确匹配 |

**拟合判定**：**完整拟合**。四层模型能无遗漏地还原这篇文章。

**残留项**：半佛的"运营签名"（"这是半佛仙人的第1961篇原创" + 抽奖暗号）不属于四层中任何一层——这是**发布层/运营层**的东西，不是写作本体的一部分。符合预期：四层模型只管写作本体。

## 拟合二：张佳伟《天涯海角，招之即来》

### 原文结构速写

五段主体，每段结构完全同构：

| 段 | 讲述者 | 触发事件 | 共同模板 | 食物 | 收束 |
|----|--------|----------|----------|------|------|
| 1 | 外婆 | 舅舅赌气出门 | 叹气→走进厨房→做菜 | 面饼+白糖 | "吃吧。" |
| 2 | 爸爸 | 我看书听不见叫喊 | 叹气→走进厨房→做菜 | 荷包蛋+酱油 | "指指：吃。" |
| 3 | 妈妈 | 爸爸打麻将不回 | 叹气→走进厨房→做菜 | 花生+黄酒 | 爸爸敲门 |
| 4 | 爸爸 | 妈妈怀孕发脾气出门 | 叹气→走进厨房→做菜 | 鱼头汤 | 妈妈出现在门口 |
| 5 | 爸爸 | 我赖床 | 叹气→走进厨房→做菜 | 泡饭+拌豆腐 | 我衣冠整齐坐桌前 |

第六段（我自己炒饭）打破同构——从"被做饭的人"变成"做饭的人"，自洽延伸。

末段（妈妈炖鸡汤+红烧肉）是最远召唤——天涯海角也能被招回来。余韵："十五年前的文了。"

### 四层拟合

| 层 | 提取内容 | 拟合质量 |
|----|----------|----------|
| 动作 | A09(同构叠加)×5 + A10(感官载情) + A11(万物拟人) + A12(短句重锤) + A14(余韵收束) | ✅ 完整，无遗漏 |
| 构件 | C10(同构叠加块)×5段 + C13(感官载情块)嵌入每段 + C14(短句重锤块)嵌入1-2段 + C11(余韵收束块)末尾 | ✅ 全文可切分 |
| 规则 | R11(感官替代抽象)✅ R12(结尾不升华)✅ R13(情感逐层递增)✅ R15(长句→短句呼吸)✅ | ✅ 四条均验证 |
| 拓扑 | T06(同构叠加弧：模板×N→余韵) | ✅ 精确匹配 |

**拟合判定**：**完整拟合**。这篇是四层模型的最佳验证样本——结构极度清晰、层次分明。

**残留项**：
- 第六段（作者自己炒饭）是对前五段同构的**变奏和突破**——它保留了"做菜"模板但改变了角色关系。这个"变奏"本身是否需要在模型中有位置？当前答案：属于 T06 的自然延伸（同构叠加弧允许最后一段变奏），不需要新对象。
- 末段"天涯海角"的远距离召唤——本质上是 C10 的极端延伸（叹气→做菜→我在天涯海角闻到），拓扑依然是 T06。

## 拟合三：张佳伟《巴黎：感谢切尔西》(体育分析文)

### 原文结构速写

这是一篇体育赛事分析，和天涯海角风格差异极大。结构：

1. 比分+事件时间线叙述
2. 战术分析（逼抢 vs 地面出球）
3. 关键人物归因（约根森失误）
4. 跳切到巴萨和利物浦的对比分析
5. 嵌入式跨界叙事（开罗出租车司机的原话）
6. 无正式收束，以引语自然收尾

### 四层拟合

| 层 | 提取内容 | 拟合质量 |
|----|----------|----------|
| 动作 | A13(古今嫁接→此处为跨界嵌入) + A12(短句重锤: "利物浦没有。") + 大白话判断(E9) | ⚠️ 部分覆盖：大量"事实陈述+因果分析"动作未被当前动作表收录 |
| 构件 | 无明显可切分的修辞构件，更接近"连续叙述流" | ⚠️ 构件边界模糊 |
| 规则 | R14(当代视角激活)松散适用 + R12(不刻意升华)✅ + R15(短句节奏)松散适用 | ⚠️ 规则适用但不如天涯海角紧密 |
| 拓扑 | 无法归入 T06/T07/T08 中任何一个——更接近"事件时间线+评论插入"的混合拓扑 | ⚠️ 拓扑表缺少"分析评论型"条目 |

**拟合判定**：**部分拟合**。

**这是本轮最重要的发现之一**：
- 张佳伟的体育文暴露了当前模型的**覆盖盲区**——偏分析/评论型的文章，其动作库主要是"事实陈述""因果归因""对比分析"，不在当前以修辞/叙事为核心的动作表里。
- 这意味着：如果四层模型要成为"通用储存模式"，需要补充**分析型/论证型动作和构件**，不能只覆盖叙事型和修辞型。
- 但同时注意：张佳伟的体育文虽然结构不同，**内核(人格)依然稳定**——大白话判断、跨界嵌入、短句节奏、开罗司机的引语(和人物向的引语阀门异曲同工)都在。这**再次证明人格是上层映射而非底层对象**——底层的动作/构件可以完全不同，但人格指向的"偏好方向"是稳定的。

## 拟合总结

| 样本 | 拟合质量 | 关键发现 |
|------|----------|----------|
| 半佛·钻石文 | ✅完整 | 四层模型对论证型强风格稿完全适用 |
| 张佳伟·天涯海角 | ✅完整 | 四层模型对叙事型强风格稿是最佳验证 |
| 张佳伟·体育文 | ⚠️部分 | 暴露分析型/评论型文章的动作及拓扑覆盖缺口 |

**跨样本发现**：
1. 四层模型对"强风格+强结构"的文章拟合极好
2. 对"弱结构+日常分析"的文章存在覆盖不足
3. 这不意味着模型方向错了——而是说**动作表和拓扑表需要扩展**，当前版本偏向修辞叙事型
