## 二、条件性对象(继承IB期,补边界)

### A2_历史铺垫

**原定义**: 回溯历史背景建立纵向坐标  
**原适用范围**: 全域可迁移  
**修订后**: 

| 维度 | 修订内容 |
|------|----------|
| **适用范围** | 分析型/评论型文章优先使用 |
| **不适用边界** | 对话体叙事、人物稿可省略,直接进入场景描写 |
| **触发条件** | 当文章需要"纵向坐标"时使用(如行业发展史、企业成长史) |
| **失败降级** | 改用"场景锚定开篇",放弃历史铺垫 |
| **验证证据** | C3(对话体)和C4(人物稿)均未使用历史铺垫,验证"不适用边界" |

### A7_隐喻承重

**原定义**: 用比喻承载抽象概念  
**原适用范围**: 全域可迁移,含变体  
**修订后**:

| 维度 | 修订内容 |
|------|----------|
| **适用范围** | 分析型/评论型文章优先使用 |
| **不适用边界** | 人物稿、对话体叙事较少使用,改用"意象回归" |
| **触发条件** | 概念抽象度高(如"真实稀缺性""企业文化")且直接解释难以产生记忆点时 |
| **失败降级** | 改用"对比案例+数据支撑"组合,放弃隐喻 |
| **验证证据** | C1"玻尿酸成精的伪人感"、C2"体育克苏鲁"均为隐喻承重;C3/C4较少使用隐喻,改用意象回归 |

### A10_共识拆解

**原定义**: 先陈述共识再用常识比喻打破  
**原适用范围**: 分析型文章可迁移  
**修订后**:

| 维度 | 修订内容 |
|------|----------|
| **适用范围** | 分析型文章优先使用 |
| **不适用边界** | 人物稿、对话体叙事不适用 |
| **触发条件** | 当文章需要"打破刻板印象"时使用 |
| **失败降级** | 改用"对比案例+反讽式对比",放弃共识拆解 |
| **验证证据** | C1"AI评论的泛滥,不是天灾,是人祸"、C2"中国品牌达到千亿体量,我都已经视为理所当然了"均为共识拆解 |

## 三、新增对象(P0级,IIA期验证确认)

### A16_反讽式对比

**定义**: 用极端反讽对比展示荒谬,揭示真相  
**最小例子**: C1第1段:"A评论'碳水爱好者闭眼冲' vs B评论'垃圾玩意,害我吃胖了五六斤'"  
**适用范围**: 半佛仙人强风格特色,适用于分析型/评论型文章  
**不适用边界**: 人物稿、对话体叙事不适用  
**触发条件**: 当需要"揭示伪人感/虚伪"时使用  
**失败降级**: 改用"对比案例+数据支撑",放弃反讽

**验证证据**:
- C1第1段: A评论vs B评论,揭示AI评论的伪人感
- C2未使用反讽式对比,验证"非必需对象"

### A17_分层递进论证

**定义**: 三个或以上层次递进,每层都更难攻破  
**最小例子**: C1第2-4章: 文本侧写(入门)→行为侧写(中级)→物理锚定(高级)  
**适用范围**: 半佛仙人强风格特色,适用于分析型文章  
**不适用边界**: 人物稿、对话体叙事不适用  
**触发条件**: 当论证需要"层层加码"时使用  
**失败降级**: 合并弱层,减少层数,保证每层证据充分

**验证证据**:
- C1第2-4章: 三层防线递进,验证"分层递进"结构
- C2第1-5章: 五层"猛"递进,验证"分层递进"变体

### A18_荒诞假设推演

**定义**: 提出荒诞假设并严肃推演,回到现实对照  
**最小例子**: C1第3章:"如果一家店今天【老板有喜休息一天】,但它的主页上又新增了一个评论"  
**适用范围**: 半佛仙人强风格特色,适用于分析型/评论型文章  
**不适用边界**: 人物稿、对话体叙事不适用  
**触发条件**: 当需要"揭示逻辑漏洞/荒谬性"时使用  
**失败降级**: 改用"对比案例+反讽式对比",放弃荒诞推演

**验证证据**:
- C1第3章: 荒诞假设推演,揭示AI评论的荒谬
- C2未使用荒诞推演,验证"非必需对象"

## 四、新增对象(P1级,IIA期验证确认)

### A19_事实锚定开篇

**定义**: 用关键事实/数据开篇,而非问题锚定  
**最小例子**: C2第1段:"安踏今天发了财报。依然和之前一样猛"  
**适用范围**: 财报/数据解读类文章优先使用  
**不适用边界**: 观点论证类文章优先问题锚定开篇  
**触发条件**: 当文章主题为"财报/数据解读"时优先使用  
**失败降级**: 事实锚定后立即引入问题锚定("为什么安踏能这么猛?")

**验证证据**:
- C2第1段: 事实锚定开篇,验证"财报类文章开篇变体"
- C1/C3/C4均用问题锚定开篇或对话开篇,验证"不同文章类型适用不同开篇"

### A20_重复锤击强化

**定义**: 同一关键词/判断重复出现,强化印象  
**最小例子**: C2全文"猛"字出现20+次,每章标题都包含"猛"  
**适用范围**: 半佛仙人强风格特色,适用于分析型/评论型文章  
**不适用边界**: 人物稿、对话体叙事较少使用  
**触发条件**: 关键判断词重复3-5次为宜,过多会削弱新鲜感  
**失败降级**: 减少重复次数,改用"数据支撑+对比案例"替代部分重复

**验证证据**:
- C2全文: "猛"字20+次,验证"重复锤击"风格
- C1未使用重复锤击,验证"非必需对象"

### A21_对比收编证明

**定义**: 用对比案例证明自己的观点  
**最小例子**: C2第4章:"同行们搞不好,救不活,但安踏做到了"  
**适用范围**: 分析型/评论型文章可迁移  
**不适用边界**: 人物稿可能不适用  
**触发条件**: 当需要"强化论证说服力"时使用  
**失败降级**: 改用"数据支撑+名人引用",放弃对比收编

**验证证据**:
- C2第4章: 同行对比,证明安踏实力
- C1/C3/C4未使用对比收编,验证"非必需对象"

### A22_对话体叙事

**定义**: 整篇文章用对话形式推进,非传统论述结构  
**最小例子**: C3全文: 人vs猫的对话,通过猫的视角揭示真相  
**适用范围**: 张佳伟强风格特色,适用于"揭示被遮蔽的真相"类文章  
**不适用边界**: 财报分析、技术解读类文章不适用  
**触发条件**: 当主题涉及"揭示被遮蔽的真相""打破常识认知"时优先使用  
**失败降级**: 对话体用于开篇和结尾,中间改用论述体+案例支撑

**验证证据**:
- C3全文: 对话体叙事,验证"对话体弧"
- C4未使用对话体,验证"非必需对象"

### A23_感官载情块

**定义**: 情感编码进食物/声音/颜色等感官元素  
**最小例子**: C3多处:"呼噜噜,呼噜噜..." - 猫的呼噜声承载满足感  
**适用范围**: 张佳伟强风格特色,适用于人物稿/叙事散文  
**不适用边界**: 分析型/评论型文章较少使用  
**触发条件**: 当需要"情感共鸣"时使用  
**失败降级**: 改用"价值观收束",放弃感官载情

**验证证据**:
- C3多处: "呼噜噜"承载情感
- IB期张佳伟样本已识别,IIA期再次验证确认

### A24_短句重锤

**定义**: 流水长句后插入极短判断句  
**最小例子**: C3中间段落:"谋生本来就辛苦喵"  
**适用范围**: 张佳伟强风格特色,适用于人物稿/叙事散文  
**不适用边界**: 分析型/评论型文章较少使用  
**触发条件**: 当需要"精准一击"时使用  
**失败降级**: 改用"数据支撑",放弃短句重锤

**验证证据**:
- C3中间段落: "谋生本来就辛苦喵"
- IB期张佳伟样本已识别,IIA期再次验证确认

### A25_意象回归收尾

**定义**: 开头画面/意象在结尾以变体再现  
**最小例子**: C4开头:"清越的笛声",结尾:"就像那一缕清越的笛声,其实无遮无拦,纯粹之极"  
**适用范围**: 张佳伟强风格特色,适用于人物稿/叙事散文  
**不适用边界**: 分析型/评论型文章较少使用  
**触发条件**: 关键意象在开头和结尾各出现1次,形成呼应  
**失败降级**: 改用"价值观收束"替代意象回归

**验证证据**:
- C4开头+结尾: 清越笛声回归
- C3未使用意象回归,验证"非必需对象"
- IB期已识别,IIA期再次验证确认

### A26_人物真相钻头

**定义**: 围绕人物的一个骨相反复回钻  
**最小例子**: C4全文: 围绕"金镶玉讨厌虚伪"这个骨相,在不同场景反复揭示  
**适用范围**: 人物稿特色对象,适用于"揭示人物本质"类文章  
**不适用边界**: 分析型/评论型文章不适用  
**触发条件**: 当主题为"揭示某个人物的本质"时优先使用  
**失败降级**: 减少钻头次数,改用"对比案例+场景描写"替代部分钻头

**验证证据**:
- C4全文: 反复回钻"金镶玉讨厌虚伪"
- IB期已识别,IIA期再次验证确认
- C3未使用人物真相钻头(非人物稿),验证"人物稿专属"

### A27_矛盾画面对比

**定义**: 用矛盾画面展示人物复杂性  
**最小例子**: C4第1段:"刚还软玉温香,忽然相思柳叶镖杀人翻脸无情"  
**适用范围**: 人物稿特色对象  
**不适用边界**: 分析型/评论型文章不适用  
**触发条件**: 当需要"展示人物复杂性格"时使用  
**失败降级**: 改用"人物真相钻头",放弃矛盾画面

**验证证据**:
- C4第1段: 矛盾画面展示金镶玉复杂性
- C3未使用矛盾画面(非人物稿),验证"人物稿专属"
