## 证据摘要

- 三篇人物向样本的开头都先给出“人物核心轴”或其等价表达，再由后文反复回钉。
- 三篇样本都存在明显公众标签或外部误读，但这些标签只作外压源，正文必须穿透它们才能成立。
- 三篇样本都不是简单时间线：真正的段落排序逻辑是“围绕人物核心轴回收材料”。

## 跨样本共证

- 开头先立轴：
  - B1 `急迫`
  - B2 `被误解后的继续滑`
  - B3 `不满 / 确定性`
- 中段反复回钉：
  - 三篇都把事件、失败、产品或伤病写成“核心轴”的购买材料
- 外部标签存在但不可冒充核心：
  - `机器人 / 天才少年`
  - `科学家父亲 / 特权`
  - `前理想总裁 / 财富自由`
- 结论：
  - 人物向稳定存在
  - 稳定项应记为 `人物核心轴`
  - 定位应是条件性对象 / 稿型子型，不是通用核心对象
