## 样本1：脑机接口门诊的一个下午

### 基本信息

| 项目 | 内容 |
|------|------|
| 来源 | 冰点周刊 |
| 类型 | 前沿技术现场报道 |
| 字数 | ~4000字 |

### 路由判断链

```
L1 通用写作物理
  ↓
L2 组合语法：M2_SETUP + M2_PROOF + M2_RESONANCE_CLOSE
  ↓
条件性对象：CO-01 单现场时间切片 + CO-02 多地域群像拼接 + CO-03 新技术临床落地
  ↓
稿型子型：GS-01 前沿科技现场报道 + GS-05 社会议题观察
  ↓
人格：冰点周刊 investigative_narrator（严肃记者隐身白描）
  ↓
修辞积木：M4_P_CINEMATIC_OPENING + M4_P_CONCRETE_OVER_ABSTRACT + M4_P_RESONANCE_ENDING
  ↓
输出：新兴技术临床落地现场群像报道
```

### 最终路由

**主路线**：新兴技术临床落地 → 单时间窗口现场群像报道

### 可复用写法路线（5条）

1. **时间切片锚定法**：选普通出诊日，精确时间，按就诊顺序记录
2. **多地域患者拼贴法**：刻意保留不同地域来源，用流动展现吸引力和门槛
3. **白描优先原理退后法**：不开篇讲原理，每段至少一个精确锚点
4. **筛选机制透明化**：呈现为什么能做/不能做，透明比只讲成功更有可信度
5. **时空收束收尾法**：同期政策事件罗列，医生一句话留白结束

## 样本2：影石十年

### 基本信息

| 项目 | 内容 |
|------|------|
| 来源 | 戴老板（饭统戴老板） |
| 类型 | 公司史+产业分析混合型 |
| 字数 | ~12000字 |

### 路由判断链

```
L1 通用写作物理
  ↓
L2 组合语法：M2_SETUP（产业概念）+ M2_PROOF（多主体）+ M2_RESONANCE_CLOSE（产业命题）
  ↓
条件性对象：人物锚点 + 供应链锚点 + 多主体共生叙事
  ↓
稿型子型：产业命题型公司史（新子型）
  ↓
人格：戴老板 产业生态解剖者
  ↓
修辞积木：历史回音壁块 + 产业概念定义块 + 多人物锚点
  ↓
输出：产业命题型公司史
```

### 最终路由

**主路线**：公司史 → 产业命题型公司史（产业中心拓扑）

### 可复用写法路线（5条）

1. **产业概念先行开篇**：用历史经典案例引出产业概念，再出当代案例
2. **拆分叙事权重**：创始人/产品占1/3，供应链合作伙伴占1/3
3. **论证相互成就**：品牌创新给供应商利润空间，供应商升级反哺产品
4. **落点产业命题**：结尾升华到中国产业红利，不局限于企业自身
5. **软硬一体总结**：硬件是载体，软件是护城河，供应链是基础

## 样本3：一位老将的谋局样本

### 基本信息

| 项目 | 内容 |
|------|------|
| 来源 | 深蓝观 |
| 类型 | 人物+行业问题混合型 |
| 字数 | ~6000字 |

### 路由判断链

```
L1 通用写作物理
  ↓
L2 组合语法：M2_SPIRAL_REVEAL（问题层层揭示）
  ↓
条件性对象：样本型评论子型 + 复盘排雷子型 + 观点社论子型
  ↓
稿型子型：样本化观点文（人物为样本，问题为主体）
  ↓
人格：深蓝观 样本化观点文人格
  ↓
修辞积木：钩子-三段证-升华树形拓扑
  ↓
输出：样本化观点社论稿
```

### 最终路由

**主路线**：人物稿 → 样本化观点文（权重配比：人物20-30%，行业问题70-80%）

### 可复用写法路线（5条）

1. **开篇钩子路线**：用最新热点事件引出人物，不是从出生写起
2. **样本剪裁路线**：按问题逻辑剪裁人生经历，每个经历产出观点
3. **权重倾斜路线**：70%篇幅给问题分析，30%篇幅给人物故事
4. **技术处理路线**：技术只讲"是什么"和"为什么有价值"
5. **终局升华路线**：落点在时代，不是人物个人

| 字数 | ~5000字 |

```
L1 通用写作物理
  ↓
L2 组合语法：M2_SETUP（双人物并置）+ M2_RESONANCE_CLOSE（意象回归）
  ↓
条件性对象：GS-01扩展（双核心轴）+ GS-06（意象回归收尾）

  ↓

  ↓

  ↓
```

### 可复用写法路线（5条）

1. **时间差双核互文法**：两代同领域女性创作者，十年时间差形成互文
2. **主题先行双人物诠释法**：先定公共议题，再选代表性人物
3. **相遇框架叙事法**：用一次现实相遇作为开篇和收尾框架
4. **植物隐喻贯穿法**：用一个自然隐喻贯穿始终
5. **商业植入主题融合法**：品牌理念与主题契合自然融入

## 样本5：乳腺癌战场上

### 基本信息

| 项目 | 内容 |
|------|------|
| 来源 | 人物（renwu账号） |
| 类型 | 人物+医疗问题深度耦合混合型 |
| 字数 | ~4500字 |

### 路由判断链

```
L1 通用写作物理
  ↓
L2 组合语法：M2_SETUP（野心命题）+ M2_PROOF（技术演进）+ M2_RESONANCE_CLOSE（野心重新定义）
  ↓
条件性对象：GS-01人物核心轴 + GS-06意象回归收尾
  ↓
稿型子型：深度耦合混合型（区别于浅度拼接型）
  ↓
人格：女性身体尊严捍卫者 + 医学边界破冰者
  ↓
修辞积木：人物野心升级曲线 ↔ 乳腺癌治疗进步曲线
  ↓
输出：深度耦合混合型人物特写
```

### 最终路由

**主路线**：人物稿 → GS-01人物核心轴 → 深度耦合混合型 → GS-06意象回归收尾

### 可复用写法路线（5条）

1. **深度耦合型**：GS-01人物核心轴 + 人物一生志业就是医疗问题 + 时间线螺旋升级
2. **浅度拼接型**：GS-01人物开篇 + 独立医疗问题阐述 + 回归人物收尾
3. **演讲体口述型**：编辑短导读 + 第一人称全程主导 + 自我命题升华结尾
4. **女性议题+医学**：女性身份共情开场 → 具体医疗实践展开 → 回归女性力量命题
5. **观念破冰型**：指出旧观念痛点 → 人物带着新技术登场 → 观念破冰过程 → 新观念收尾
