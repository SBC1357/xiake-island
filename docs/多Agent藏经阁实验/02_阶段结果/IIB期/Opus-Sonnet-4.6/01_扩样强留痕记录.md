## S1：脑机接口门诊的一个下午

### 1. 主人格

**P05 witness-correspondent / 在场记录者**

作者以冷静近距观察者身份在场，不介入、不评论，用场景和细节说话。与 P01(witness) 最接近，但 P01 偏向有主角的长线叙事见证，P05 偏向无主角的群像现场。情绪基调 = 克制 + 温含同情但不煽情。偏好 {场景承载、细节买权、多人物穿插}；排斥 {宏大判断、数据锤击、政策推演、主观抒情}。

### 2. 主拓扑

**TN-05 现场辐射弧 / Scene-Hub Radial Arc**

一个固定时空锚点（协和脑机接口门诊的一个下午）作为中心，多个患者故事作为辐射臂依次展开，每个臂带出脑机接口的不同面向（技术现状、适用条件、治疗效果、经济门槛、患者心理、政策进展）。不是级联递进，不是并列推进，而是从一个中心向外辐射。

结构拆解：
- **锚点**：3月5日下午1点，协和医院脑机接口门诊
- **辐射臂1**（葛雷）：远道求医、精细动作恢复受限 → 暴露技术边界
- **辐射臂2**（罗娟娟）：肌张力过高 → 暴露适用门槛
- **辐射臂3**（黄子逸）：经济门槛 → 暴露可及性问题
- **辐射臂4**（乔伊）：成功案例、运动想象机制、闭环训练原理 → 核心技术解释 + 患者情感弧
- **锚点回收**：政策写入政府工作报告、标准制定、未来可及性

### 5. 逼出的新边界

1. **压力起手不是通用对象**。这篇文章用「平淡入场」而非「张力起手」——现场报道型稿件不需要制造开篇悬念，而是用在场感拉入读者。这意味着 U02（开篇必须制造认知/视觉张力）在报道型稿件中需要降级。
2. **关键问题钉子可以是隐式的**。全篇没有单一句子把问题钉住，但读者读完后回看，问题清晰。这意味着「钉子」可以是文本外的、由多个场景合力构成的。
3. **现场辐射弧是新拓扑**。既不是级联、也不是并列、也不是双层递进。它是一个时空锚点 + 多辐射臂的结构，在现场报道/纪实类文章中稳定出现。
4. **多人物穿插是新的结构推进动作**。不是讲述者交接（A03），因为叙事视角始终是作者/在场者；但人物在进出之间推进了对问题的理解。

### 6. 新对象候选

- **DN-01 平淡入场 / Quiet Entry**：以场景还原方式开篇，不制造张力，靠在场感吸引读者
- **DN-02 多人物穿插推进 / Multi-Character Interleave**：多个人物依次出场并退场，每个带出一个新面向
- **TN-05 现场辐射弧 / Scene-Hub Radial Arc**

### 7. 新规则候选

- **RN-08 报道型稿件不强制压力起手**：现场报道 / 纪实类稿件中，U02 降级为可选
- **RN-09 隐式钉子规则**：问题钉子可以不在文本中显式出现，但必须能被场景合力回溯还原

### 8-9. 触发条件与失败降级

- RN-08 触发：文章结构是「现场/场景驱动」而非「论证/叙事驱动」时触发
- RN-08 降级：如果场景开篇后 500 字内仍无法让读者感到「在场」，降级回张力起手
- RN-09 触发：文章无显式问题句但存在主题统一性时触发
- RN-09 降级：如果隐式钉子回溯不出来，说明文章缺乏主题焦点，需要补显式钉子

### 10. 处理结论

**条件通过（需补拓扑 + 动作）。** 当前框架能吸收这篇，但暴露了 3 条新边界：压力起手/问题钉子的非强制性、现场辐射弧拓扑、多人物穿插推进动作。这些不推翻旧骨架，而是在「现场报道型」稿件中需要补条件分支。
