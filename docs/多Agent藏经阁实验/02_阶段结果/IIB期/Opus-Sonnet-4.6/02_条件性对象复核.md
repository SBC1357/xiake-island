## 三、对象层级总览（IIB 更新后）

```
核心稳定层（所有稿型必须检查）
├── 细节买权                    5/5 ← 最稳定
├── 失败收编                    5/5
├── 历史回望链                  4/5 + 微型1
├── 关键问题钉子                4/5显式 + 1隐式
├── 边界后拔高                  5/5（含演讲稿变体）
│
条件稳定层（稿型匹配时必须检查）
├── 压力起手                    ← IIB 从核心降级（报道/演讲型不适用）
├── 解释桥                      ← IIB 从核心降级（非知识型不适用）
├── 比喻承重                    ← IIA 已降级，IIB 维持
├── DA-06 权威锚定              ← IIB 升级（3/5命中 + 新变体）
├── RN-03 蜕变链                ← 有成长弧时触发
│
条件层（特定稿型才检查）
├── DA-01 数据锤击
├── DA-04 案例递进
├── DA-05 转折节点钉子
├── DA-08 风险解剖
├── RN-04 第三方背书
├── DN-01~DN-07 (IIB新增动作)
├── RN-08~RN-14 (IIB新增规则)
│
保留层（孤证，待更多样本确认）
├── TN-05 现场辐射弧            ← 1/5命中
├── TN-06 双线交织弧            ← 1/5命中
├── P05 在场记录者              ← 1/5命中
```

## 四、结论

1. **核心稳定层有 2 个对象降级**：压力起手、解释桥从核心降为条件稳定。这不是推翻旧骨架，而是精确化适用域。
2. **IIA 的降级判断获得确认**：比喻承重在 IIB 中继续表现为条件性，验证了 IIA G03 降级的正确性。
3. **权威锚定升级**：DA-06 从条件升级为条件稳定，并出现产业案例类比的新变体。
4. **新增 7 个动作、2 个拓扑、7 条规则、1 个人格**。这些都是边界扩展，不是骨架重构信号。
5. **框架骨架（动作+构件+规则+拓扑 四层 + 人格映射上层）保持稳定**，IIB 的贡献是精确化条件分支和扩展稿型覆盖面。
