## 一、IIB 期四个核心问题的回答

### Q1：骨架需不需要结构性重组，还是边缘修补就够？

**边缘修补就够。**

四层骨架（动作 + 构件 + 规则 + 拓扑）+ 人格上层映射的架构在 IIB 5 篇扩样中没有出现崩溃信号。5 篇样本覆盖了 3 种高风险混合类型（无主角群像报道、公司史+产业分析、人物+行业分析），以及 2 种格式异常（品牌合作稿、演讲稿），框架均能给出可行路由。

需要做的是：
- 2 个核心对象（压力起手、解释桥）从核心层降级为条件稳定层
- 增加 7 个条件性动作、2 个拓扑、7 条规则、1 个人格
- 为演讲稿 / 第一人称自述补格式分支
- 为品牌合作稿补标注规则

这些全部是「在现有骨架上补条件分支」，不是结构重组。

### Q2：哪些条件性对象可以稳定？

**稳定的（证据充分）：**
- 细节买权：5/5，最稳定的核心对象
- 失败收编：5/5
- 历史回望链：4+1/5
- 关键问题钉子：4+1/5（含隐式变体）
- 边界后拔高：5/5（含演讲稿变体）
- DA-06 权威锚定：3/5，升级为条件稳定

**需降级的（适用域比预期窄）：**
- 压力起手：3/5，报道型和演讲型不适用 → 从核心降为条件稳定
- 解释桥：3/5，非知识密集型稿件不必须 → 从核心降为条件稳定

**维持条件的（证据不够强化也不够否定）：**
- 比喻承重：2/5 贯穿，维持 IIA 的条件降级判断
- DA-01 数据锤击：弱
- DA-04 案例递进：仅公司史命中
- DA-08 风险解剖：弱

### Q4：能不能为每篇样本生成 ≤5 条可行写法路线？

**全部能。**

- S2：4 条路线，路线 A（品类定义标杆叙事）为主
- S3：4 条路线，路线 A（人物决策链+行业判断）为主

主路线选取理由均在 `03_路由判断记录.md` 中给出。

## 二、补充样本触发判断

### 触发条件回顾（来自实验方案）

> 在以下条件出现时触发补充样本：
> 1. 主样本中出现 ≥2 个结构性争议（不是边界模糊，而是框架无法调和）
> 2. 路由在 ≥2 个样本上完全失败（给不出可行路线）
> 3. 某个核心对象在 ≥3 个样本上失效

### 本中心判断

**不触发补充样本。**

理由：
1. 结构性争议 = 0。S1/S3/S5 需要补条件分支，但这不是「框架无法调和」，而是「需要扩展覆盖面」。框架的四层骨架从未被某篇样本推翻。
2. 路由失败 = 0。5/5 样本均给出了可行主路线和替代路线。
3. 核心对象失效 = 0。压力起手和解释桥虽然降级，但「降级为条件稳定」不等于「失效」。它们在适用的稿型中仍然有效。

> 注意：S4（品牌合作）和 S5（演讲稿）确实是格式异常，但异常指向的是「稿型分类需要扩展」而非「框架骨架失效」。补充样本（刘家琨建筑手记/蔡磊Biotech病叙事）是为了验证更极端的边界，在本中心的分析中不需要触发。

### 骨架完整性评估

```
四层骨架     IIA判断           IIB判断             跨期趋势
─────────   ─────────────     ─────────────       ──────────
动作层       8基础+14IB+13IIA   +7IIB条件性动作     持续扩展，核心收敛
构件层       稳定               稳定               无变化
规则层       G01-06+U01-04+    +7条条件性规则      持续扩展，2条降级
            RN01-07
拓扑层       T01-08+TN01-04    +2条件性拓扑        持续扩展
人格层       P01-P04            +P05条件性          持续扩展
```

### 整体判断

**支持当前骨架，继续边缘精化。**

IIB 期的贡献是：
1. 确认 IIA 的骨架稳定性判断——5 种新稿型 / 格式均未推翻四层骨架
2. 精确化条件分支——压力起手/解释桥从核心降级、权威锚定升级、比喻承重维持条件
3. 扩展覆盖面——补了群像报道、公司史、人物+行业、品牌合作、演讲稿 5 种稿型的路由
4. 路由能力验证——5/5 样本均可路由，说明框架具备实用的写作指导能力

### 建议下一步

- 如果进入 III 期，可以测试补充样本（刘家琨/蔡磊/潮新闻），但预期不会推翻骨架
- 更有价值的下一步是：将框架从「分析能力」转向「生成能力」——即用框架实际生成一篇文章，测试其可操作性
- 条件性对象目前共 40+ 个，需要做一次「对象压缩」，合并功能重叠的对象
