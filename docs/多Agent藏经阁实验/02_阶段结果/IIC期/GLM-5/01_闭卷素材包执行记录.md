## 一、执行概述

本轮执行5道闭卷测试题：A类校准卷、B类半结构卷、C类实战卷、D类压限卷、附加仿写对照题。

执行原则：
- 不编造素材之外的事实
- 不依赖作者身份、刊物名、原标题等明牌信息
- 自主完成结构组织和主次取舍

## 二、各卷执行记录

### A类校准卷：养老院的春节"喘息服务"

**素材特征**：高整理度、强卡片化，包含事实卡、引语卡、关系卡、时间线、冲突点、任务要求、噪音卡。

**执行策略**：
- 选择从现场场景切入（养老院院子里的老人），后文带出更大制度和关系问题
- 结构：现场感 -> 短托服务 -> 团圆压力 -> 安全与自主边界 -> 新型城市生活缝隙
- 主动舍弃部分噪音卡信息（写春联包饺子等活动细节），只保留能支撑主线判断的材料
- 重点处理"问题现场群像稿"要求：不只写单一人物，也不写成纯行业分析

**关键判断**：
- 主承重信息：短托服务、照护压力、老人意愿边界、机构安全逻辑
- 次要信息：具体活动细节、个别老人的小愿望（作为现场感点缀保留）
- 噪音处理：舍弃营销式活动描述，保留有判断价值的关系张力

**字数**：约2000字

### B类半结构卷：皮毛市场兴衰

**素材特征**：中等提示强度，提供事实、时间信息和少量人物线索，但不提供现成的冲突总表。

**执行策略**：
- 自主组织"神话、回落、转身"之间的关系
- 结构：淘金年代 -> 行业放缓 -> 两条新路 -> 两代人差异 -> 不确定中的确定
- 从行业变化角度切入，但保持人物线索的穿插
- 自主判断重心：不是只写怀旧，也不是只写电商分析

**关键判断**：
- 主承重信息：鼎盛期数字、行业放缓、两条新路尝试、两代人差异
- 次要信息：夜市摆摊细节、具体玩具未买成等个人回忆
- 噪音处理：舍弃具体颜色款式细节，保留有判断价值的时间线和数字

**字数**：约1900字

### C类实战卷：谣言击中小餐饮店

**素材特征**：碎片化材料，顺序无意义，不预设主次，接近真实处理后素材池。

**执行策略**：
- 自主判断文章类型：个案+机制混合稿
- 从一家店下刀，再向外扩到谣言的固定链条和社会机制
- 结构：个案现场 -> 谣言传播 -> 固定链条 -> 追不到的源头 -> 为什么能反复得手 -> 污名内化 -> 谣言换壳能力 -> 熟人网络扩音
- 自主识别并舍弃噪音：具体小吃种类、精确时间点、截图细节

**关键判断**：
- 主承重信息：谣言传播链条、污名内化、熟人网络扩音、追不到源头
- 次要信息：具体产品名称、直播平台细节
- 噪音处理：舍弃产品营销信息，保留社会机制分析的核心材料

**字数**：约2500字

### D类压限卷：女性友谊关系

**素材特征**：接近C类材料强度，额外加入写法限制（人格、主拓扑、情绪温度等）。

**执行策略**：
- 遵循附加限制：温柔说书人、多组镜像人物串联、中高情绪温度、中证据密度
- 结构：一组深写（第一组）+ 两组辅助回声（第二组、第三组）+ 差异照亮判断
- 不写成女性励志宣言，不平均切块，不喊口号
- 从具体关系出发，轻轻落回人与人之间的照亮感

**关键判断**：
- 主承重信息：三组朋友的具体互动、误解与重新靠近、差异变成照亮
- 次要信息：征集背景、品牌口号
- 噪音处理：舍弃营销式口号、产品信息，保留关系细节

**字数**：约2100字

### 附加仿写对照题：人物特稿

**素材特征**：介于B类与D类之间的处理强度，额外加入写法限制。

**执行策略**：
- 遵循附加限制：冷静分析师人格、单人物核心轴+多轮问题回钉、中低情绪温度、高证据密度
- 核心：把"对不合理的强烈反应和对确定性的偏执"作为人物核心轴
- 行业、产品、环保、空间等材料都成为人物轴的反证，不平铺并列
- 不写成企业宣传文，不只堆产品功能

**关键判断**：
- 主承重信息：人物核心轴（不满意、追问、确定性偏执）、行业判断、环保实验室、睡眠场景
- 次要信息：产品型号、参数数量
- 噪音处理：舍弃营销口号、产品命名细节，保留人物判断逻辑

**字数**：约2400字
