# 他受不了“不合理”，于是想把家做成一件确定的产品

状态：Completed  
日期：2026-04-04  
题型：IIC 第一轮附加仿写对照题

在他看来，今天的人可以在一辆车里享受相当前沿的科技，却还得在家里忍受高度落后的体验。这件事本身就不合理。传统家装像一场被默认接受太久的“攒机式”拼装：品牌是分散的，设备是各管各的，渠道层层叠叠，成本并不透明，最后交到用户手里的也不是一个完整结果，而是一堆勉强凑起来的部件。正是这种时代错位，把一个曾在新能源行业做到高位的人，又一次推回到一号位。

他几乎对什么都不满意。这不是情绪化发作，而更像一种根深蒂固的工作方式。碰到“伪科技”“体验割裂”“结构不合理”这种事，他会本能地起反应，否定力度强得近乎口头禅，但停不下来的从来不是抱怨，而是追问：为什么会这样，能不能彻底改掉，怎样才能把结果做成确定的。精力旺也是这种性格的外溢。长会结束，别人已经疲了，他还可能继续兴奋地讲细节、指现场、拆问题。对很多人来说，这种长期高压近乎折磨；对他来说，它反而接近一种必须维持的工作浓度。

这种对“不确定”的警觉，并不是转到新赛道以后才长出来的。早年在外企做高管时，他已经很强烈地感受过时代周期带来的摇晃，于是跳去更有发展前景的新赛道。新能源汽车行业的多年经历，让他第一次把“确定性”做成了看得见的结果。可一旦习惯了结果可控，他对失控就更敏感。二零二二年前后，一次腿伤把他按在家里，迫使他长时间正面感受装修和居住体验里的种种荒诞。那不是一次温和的观察，而像是把他本来就压着的火，一下子点明了。

他先算账。这个行业规模足够大，但传统渠道、租金、获客和返点层层吃掉了成本。一个普通家庭花几十万元装修，最后真正落在自己生活质量上的那部分，远低于表面账单。然后他再往下拆。传统交付更像原材料和零散设备的堆砌，不是一个完整产品；所谓智能，常常只是设备各管各的假智能；看起来热闹的功能，解决不了最底层的可用性和一致性。问题一旦被他这样拆开，方向也就跟着清楚了。用户买的不是零件，而应该是一个确定结果。

所以他的定义，不是卖若干智能设备，而是直接交付一个软硬件一体化的空间产品。内部排序也很死板，先健康，再舒适，再智能。这个顺序几乎不容讨论，因为在他看来，底线一旦没守住，后面的花哨都只是遮羞布。老人会不会用，孩子会不会用，自己家愿不愿意装，是所有方案都得先过的三道关。凡是过不了，他就不认。

睡眠场景最能说明他的判断。他拿过一组卧室实测数据，看到密闭环境下，某项空气指标在半夜升到远高于健康底线的水平。这个细节在他那里不会只停留成“又一个痛点”，而会立刻被推回同一结论：传统家装根本没有为最终健康结果负责。于是他的解法不是再叠一台设备，而是让传感器、算法、通风和温控协同工作。在他描绘的理想场景里，入睡、深睡、换气、唤醒都应尽量无感完成。所谓真智能和伪科技的分水岭，也正落在这里。不是功能多不多，而是系统有没有把结果管起来。

同样的控制欲还延伸到空间本身。他对空间利用极度敏感，觉得很多家庭真正浪费的不是面积，而是体积，是那些看上去体面、实则没有功能的“木头盒子”。为了把不同系统塞进更顺手、更省空间的结构里，他会逼着团队、供应商和工艺一起改，某些模块甚至做出多个变体版本。美感在他的逻辑里也不是贴上去的装饰，而是功能和结构打通之后自然长出来的秩序。换句话说，他连审美都要求服从结果，而不是拿来遮盖问题。

最不能妥协的一块是环保。行业里很多供应商愿意保证单一材料，却不愿意为最终叠加结果负责。他要的偏偏就是后者。所有材料组合进真实空间以后，结果仍得达标，甚至还要给用户后续自购物品预留余量。为了这件事，公司早期就砸钱搭实验室，做大量交叉测试，被淘汰的材料远多于能留下的材料。更极端的时候，他甚至停掉过利润不错的配套业务，只因为量产结果和送测样品不一致。在这种决断里，他最在意的从来不是损失有多大，而是有没有什么环节把“确定性”偷走了。

成为一号位以后，他更累，也更兴奋。那种近乎强迫式的控制，不只是工作风格，也是他理解世界的办法。可他并不把自己讲成改变时代的英雄。他承认，个体很多时候只是踩中了时代窗口。只是即便如此，他仍愿意把未来十年甚至更长时间押在一件事上：把居住体验真正推进一代。这个愿景听上去很大，但落回他身上，仍然是一以贯之的同一种反应。看见不合理，他就不能算了；看见结果不确定，他就非得一路追到底。也正因为这样，他做的并不只是一个空间产品，而是在试图把自己最受不了的那种混乱，改造成一套可以被计算、被控制、被验证的生活秩序。
