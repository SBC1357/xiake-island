# 01_闭卷素材包执行记录

执行中心：Opus/Sonnet 4.6  
日期：2026-04-04  
阶段：IIC 第一轮

## 执行顺序

按 A → B → C → D → 附加题 顺序执行，逐卷读取素材包后直接成稿。

## 各卷执行概况

### A类 校准卷

- 素材包：高整理度卡片化素材。事实卡 20 条、引语卡 8 条、关系卡 6 条、时间线 6 条、冲突点 6 条、噪音卡 6 条
- 处理方式：从一个具体的机构现场起笔，依次拉出照护家属的喘息需求、短托服务的商业边界、老人意愿与家庭安排的张力，最后收到更长期的照护制度缺口
- 噪音处理：噪音卡 1（春联包饺子活动）舍弃、卡 3（补贴入住）舍弃、卡 4（子女陪住）舍弃；卡 2（阳台种花）和卡 6（新年愿望细节）保留，用于支撑"老人意愿"段落；卡 5（照护者个人经历）未作为独立段落展开，融入家属视角
- 字数：约 2400 字
- 冒烟判断：系统成功调用了"从局部场景起笔 → 展开关系与制度问题"的基本骨架

### B类 半结构卷

- 素材包：中等提示强度。给出起点/鼎盛期/变化/新路子/人物与关系/引语/噪音共 8 组材料，不给冲突总表
- 处理方式：自行梳理出"淘金 → 退潮 → 转身"的三幕结构，以一个南下家庭为情感锚点，穿插两条新路子（小件周边 / 直播间）作为行业转型的具体载体
- 冲突组织：自行提炼"旧生意靠胆量和体力 vs. 新消费靠渠道和产品重定义""嫁接而非替代"
- 噪音处理：10 元玩具舍弃、夜市摆摊收入弱保留（一句带过）、春节少数开门舍弃、小件款式颜色舍弃
- 字数：约 2300 字

### C类 实战卷

- 素材包：接近真实素材池的碎片化材料，30 条不分主次、不保证顺序，另有 4 条噪音
- 处理方式：从"一家店被一句话击垮"的个案切入，逐层展开 → 当事人家庭受损 → 同类谣言链条复现 → 辟谣成本与传播结构 → 病耻感 + 性道德想象 + 熟人网络的系统性原因
- 结构判断：选择"个案 + 机制"混合稿形态
- 噪音处理：具体小吃品类、精确年份月份、截图视频表述细节、围观者情绪化发言——全部舍弃
- 主次判断验证：碎片 25（"接住流量"建议）和碎片 26（受害者荒诞感）被判断为高价值节点，用于揭示"辟谣本身的二次伤害"
- 字数：约 2800 字

### D类 压限卷

- 素材包：23 条碎片化材料 + 6 类写法限制 + 4 条噪音
- 写法限制执行情况：
  - `人格`：温柔说书人 → 全文使用平稳、不紧不慢的叙述节奏，不喊口号
  - `主拓扑`：多组镜像人物串联 → 第一组深写（职场稳定 vs. 漂泊自由），第二组中深（婚育 vs. 事业），第三组轻写（育儿焦虑 vs. 无母职玩心），最后回到统一的关系判断
  - `情绪温度`：中高 → 有温度但克制，拉黑段落保留刺痛感，不用大段抒情
  - `证据密度`：中 → 以细节（视频电话、拉黑、午休吃饭）买权，不靠抽象论述
  - `收尾方式`：从具体关系 → 普遍经验判断 → 落回人与人之间的照亮感
  - `禁用项`：未出现品牌/产品/征集活动/女性励志宣言/平均切块/大段空泛抒情
- 噪音处理：征集背景、品牌口号、补充举例细节、消费产品信息——全部舍弃
- 字数：约 2600 字
- 重点观察：`双人物共享核心轴`——第一组和第二组的处理方式不同，前者是互补型共享轴，后者是冲突-修复型共享轴；结构上未分裂，回到同一个底层判断

### 附加题 仿写对照

- 素材包：9 组材料（起手/核心轴/经历/行业判断/解决方案/睡眠传感/空间功能/环保实验/一号位判断） + 6 类写法限制 + 4 条噪音
- 写法限制执行情况：
  - `人格`：冷静分析师，但人物核心轴强显性 → 以克制笔法写人，但"不满 + 确定性"轴贯穿全文每一段
  - `主拓扑`：单人物核心轴 + 多轮问题回钉 → 行业判断、睡眠场景、空间利用、环保——每一块材料写完都拉回人物的"不满"和"确定性"
  - `情绪温度`：中低 → 有判断力和硬度，不冷冰冰
  - `证据密度`：高 → 密闭睡眠空气数据、成本账、淘汰材料比例、停配套业务——用场景和数据持续买权
  - `收尾方式`：从人物当下控制力 → 更长时间尺度的行业愿景 → 不喊口号
  - `禁用项`：未写成企业宣传文/产品功能堆砌/人物-问题分拆/空泛热血
- 噪音处理：产品线名称型号、空间营销命名、算力模块参数、时代口号——全部舍弃
- 字数：约 2700 字
- 重点观察：人物核心轴是否被写散 → 本稿以"对不合理的强烈反应 + 对确定性的偏执"为单一核心轴，所有行业/产品/环保材料均作为该轴的反证回钉，未出现人物-问题脱钩
