## C类 实战卷 测试结果

### 成稿：`12_C类_实战卷.md`

### 结构判断

成稿选择"个案 + 机制"混合稿形态，从一家店的遭遇下刀，逐层向外扩展到同类谣言链条、辟谣成本结构、传播机制和病耻感的社会根源。不是纯个案稿，也不是纯制度通报。

### 自主信息分层

素材池 30 条碎片不分主次。成稿自行完成了以下主次判断：

**主承重（占正文主体）：**
- 一家店从排队到空场的戏剧性转折（碎片 1/4/8）
- 谣言传播的熟人社会路径（碎片 6/7/9）
- 家庭被全面波及（碎片 14/29）
- 辟谣成本远高于传播成本（碎片 12/13/21/22/23）
- 病耻感 + 性道德想象 + 食品接触恐惧的三根弦（碎片 27/28）
- "接住流量"的二次伤害（碎片 25/26）

**辅助承重（用于补强论证）：**
- 同类案例的重复出现（碎片 3/18/19/20）
- 官方科普 vs. 恐惧本能的脱节（碎片 16/17）
- 信任流失的单向性（碎片 24/30）

**舍弃：**
- 具体小吃品类
- 精确年份月份
- 截图视频表述细节
- 围观者情绪化发言

### 社会解释力

成稿未止步于"谣言害人"。它提供了三层解释：

1. 为什么这种谣言能穿透：病耻感 + 性道德想象 + 食品接触恐惧同时发力
2. 为什么辟谣不好使：行为在恐惧驱动下先于信息抵达
3. 为什么责任追到最后会散掉：熟人社会的扩音效应 + 无明确源头的传播结构

### 污名处理

未写虚。通过两个具体细节钉住：（1）孩子被隔离——"有人叮嘱别的小孩不要跟他的孩子一起玩"；（2）受害者内化羞耻——"他开始觉得别人都在用异样的眼光看自己"。

### 可读性

可读。碎片化素材被重新组织为递进式结构，没有写成碎片堆砌。

### C类小结

在接近真实素材池的材料中，系统能够自主完成主次判断、选择合适的稿件形态（个案 + 机制混合）、提供足够的社会解释力，且未失去人的刺痛感。

## D类 压限卷 测试结果

### 成稿：`13_D类_压限卷.md`

### 附加限制执行

| 限制项 | 要求 | 执行结果 |
|--------|------|----------|
| 人格 | 温柔说书人 | 全文语调平稳、不紧不慢，以叙述者视角展开，无喊口号、无急促 |
| 主拓扑 | 多组镜像人物串联，允许一组深写两组辅助 | 第一组（职场 vs. 漂泊）深写，第二组（婚育 vs. 事业）中深写（含冲突-修复弧），第三组（育儿 vs. 无母职）轻写；全文回到"差异是连结的来源" |
| 情绪温度 | 中高，有温度不过热 | 拉黑段落保留刺痛感，视频电话段落有温度，未出现大段抒情 |
| 证据密度 | 中，靠细节买权 | 长视频电话、拉黑事件、午休吃饭/医院陪护、"扯闲篇像重型武器"——以日常细节为主要证据 |
| 收尾方式 | 具体关系 → 普遍经验 → 照亮感 | 末段从三组具体关系收到"照到我自己照不到的地方"，最后落回"最具体的场景" |
| 禁用项 | 不写品牌/征集/励志宣言/平均切块/空泛抒情 | 未出现任何违禁项 |

### 重点观察：双人物共享核心轴

D类成稿包含 3 组"双人物"结构，每组都是共享核心轴的一种变体：

1. **互补型**：稳定 vs. 漂泊 → 核心轴是"不同路径提供不同证据"
2. **冲突-修复型**：婚育 vs. 事业 → 核心轴是"痛苦峰值撞击后的重新理解"
3. **日常平行型**：育儿焦虑 vs. 无母职玩心 → 核心轴是"不以身份定义彼此"

三组都回到同一个底层判断："差异不是障碍，而是连结的来源。"共享核心轴未分裂。

### 结构家族接近度

成稿的结构家族特征：
- 多组镜像串联，不平均切块
- 以细节而非论断买权
- 温度持续但克制
- 收尾从具体回到普遍再落回具体

是否达到"逼近原稿家族"：在不知道原稿的前提下，成稿展现出"温柔、具体、彼此映照"的气质家族。与素材包附加限制的 6 类要求均吻合。

### 噪音处理

征集背景、品牌口号、补充举例末尾材料、消费产品信息——全部舍弃，未进入正文。

### 可读性

可读。三组故事之间的过渡自然，未出现跳跃感。

### D类小结

在接近实战卷的材料复杂度基础上，附加 6 类写法限制后，系统能够：
1. 遵守全部限制要求
2. 自主选择"一组深写 + 两组辅助回响"的拓扑
3. 将三组双人物共享核心轴收束到统一判断
4. 未出现抗过拟合失败（未靠记忆式模仿或照搬显性口号）

## 附加题 仿写对照 测试结果

（按实验方案要求，附加题结果并入本文件最后一节。）

### 成稿：`14_附加题_仿写对照卷.md`

### 人物核心轴

成稿将人物核心轴锚定为"对不合理的强烈反应 + 对确定性的偏执"。这条轴从第一段起就被建立（"他几乎对什么都不满意"），此后每一段材料展开都回钉到这条轴上：

| 材料模块 | 回钉方式 |
|----------|----------|
| 早年外企经历 | 不确定感驱使他跳赛道 |
| 新能源行业经历 | 第一次把确定性做成结果 → 渴求更强 |
| 腿伤居家 | 不满被强制放大 → 看到居住体验的荒诞 |
| 行业成本账 | "不合理"的典型载体 |
| 解决方案定义 | "确定的结果"是产品定义的核心 |
| 睡眠场景 + 传感 | "真智能 vs. 伪科技"的分水岭 = 确定性的具体化 |
| 空间利用 | 偏执到功能密度级别 |
| 环保与实验室 | 不信运气，只信算出来的确定性 |
| 一号位时代判断 | 不把自己当英雄，但愿意押十年 |

未出现"人物核心轴写散"的问题。行业、产品、环保等材料均作为核心轴的反证，不是并列平铺。

### 附加限制执行

| 限制项 | 要求 | 执行结果 |
|--------|------|----------|
| 人格 | 冷静分析师，核心轴强显性 | 克制笔法，但"不满 + 确定性"轴在每一段都有显性体现 |
| 主拓扑 | 单人物核心轴 + 多轮问题回钉 | 行业/产品/环保/空间均回钉到人物轴，不平铺并列 |
| 情绪温度 | 中低 | 有硬度和判断力，不冷冰冰，也不热血 |
| 证据密度 | 高 | 密闭空气数据、成本账、淘汰材料比例、停配套业务——持续用场景和数据买权 |
| 收尾方式 | 人物当下 → 长时间尺度愿景 → 不喊口号 | 从"把居住体验推进一代"到"没有退路"，收束在行为逻辑上 |
| 禁用项 | 不写企宣/产品堆砌/人物-问题分拆/空泛热血 | 全部遵守 |

### 仿写对照 控制精度判断

- 未把人物稿写成普通行业解释稿：行业材料全部挂在人物轴上
- 未把人物核心轴写丢：从开头到结尾，"不满 + 确定性"轴持续存在
- 未把人物-问题混合回钉写成松散拼接：每一块材料展开后都有明确的回钉句

### 附加题小结

控制精度可接受。人物核心轴强显性，多轮回钉结构完整，未出现 IIC 方案列举的三种失真中的任何一种。
