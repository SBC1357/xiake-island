## 一、C类测试结果：稀素材提炼卷

### 1.1 测试目标

核心问题：**会不会撑？**

重点观察：
- 判断提炼
- 硬过渡
- 背景调用边界
- 少料成稿能力

### 1.2 成稿分析

**素材使用情况**：

素材池共12条，偏薄。成稿使用全部12条素材，无外部编造。

**判断提炼情况**：✓ 有明确主判断

提炼出的核心判断：这类谣言能反复得手的原因是**病耻感、饮食恐惧和熟人传播三者的叠加**，伤害来自一群人的无意识合谋而非单一坏人。

这个判断贯穿全文，从个案切入后逐步展开机制分析，最终回到个案的刺痛感。

**硬过渡情况**：✓ 过渡合理

文章结构：
- 个案切入（小店被谣言击中）
- → 机制展开（为什么这类谣言总能得手）
- → 社会心理根源（病耻感、饮食恐惧、熟人传播）
- → 回到个案刺痛感

过渡通过分析推进而非空话衔接，没有硬撑。

**背景调用边界**：✓ 控制在合理范围

- 调用了"病耻感"、"饮食恐惧"、"熟人社会传播"等背景知识
- 这些背景知识服务于解释机制，没有超出素材隐含的范围
- 没有编造外部事实或引入无关背景

**少料成稿能力**：✓ 能撑起来

在素材偏薄的条件下，通过：
- 深挖个案细节（店主自证、帮工被波及、孩子被卷入）
- 展开机制分析（谣言传播链条、追查困境）
- 深挖社会心理（为什么总能得手）

把文章撑到约1600字，符合1400-1900字要求。没有用空话撑长。

## 二、D类测试结果：单点起稿卷

### 2.1 测试目标

核心问题：**会不会推开？**

重点观察：
- 议题判断
- 作者积累替代能力
- 从一点外推到一篇文章的能力

### 2.2 成稿分析

**素材使用情况**：

素材池：核心判断 + 10条锚点，极薄。

成稿组织方式：
- 深写一组关系（阿梅和小夏，婚育与事业、母职与非母职）
- 其他素材做回声（稳定与自由那组、标签之前的相遇）
- 不平均切块所有案例

**议题判断情况**：✓ 判断明确

坚持核心判断：差异不会削弱连结，反而成为彼此看见、理解和接住对方的入口。

全文围绕这个判断展开，没有偏离或稀释。

**推开情况**：✓ 能推开

从一个核心判断往外推开：
- 具体化为阿梅和小夏的故事（婚育/事业差异）
- 扩展到另一组朋友（稳定/自由差异）
- 回声：标签之前的相遇、不需要断舍离的朋友

在边界内推开，没有超出素材隐含范围，没有胡编乱造。

**边界控制情况**：✓ 控制在指定边界

题目指定的边界：
- 稳定与自由 ✓
- 婚育与事业 ✓
- 母职与非母职 ✓
- 差异如何成为映照而不是威胁 ✓

全文只在这些边界内展开，没有超出。

### 2.3 限制条件落住情况

| 限制条件 | 落住情况 |
|---|---|
| 不得编造事实 | ✓ 全部素材来自题包 |
| 只在指定边界内推开 | ✓ 控制在婚育/事业、母职/非母职等边界 |
| 不写成女性励志宣言 | ✓ 避免励志口号 |
| 不平均切块所有人物 | ✓ 深写一组，其余做回声 |
| 不用空泛抒情代替细节 | ✓ 有具体故事和细节 |
| 选定主判断其余材料服务它 | ✓ 全部素材服务于核心判断 |

### 2.4 开放性要求响应

| 开放性要求 | 响应情况 |
|---|---|
| 温柔但不煽情 | ✓ 克制表达，没有煽情 |
| 不太像散文有叙事推进 | ✓ 有明确叙事和判断推进 |
| 写出映照感 | ✓ 差异成镜子，照亮彼此 |

### 2.5 核心问题回答

**会不会推开？—— 会推开。**

从核心判断往外推开，具体化为故事，控制在边界内，没有胡编乱造或套话扩写。

### 2.6 存在的问题

- 字数略低（约1300字），接近下限1200字
- 可再深写一组关系的细节
- 结尾映照感可再强化

## 五、C-D类特殊发现

1. **稀素材提炼的关键在于判断提炼**：不是把每个面向都写满，而是提炼一个主判断，让所有素材服务于它。

2. **单点推开的关键在于具体化**：从抽象判断到具体故事，再从故事回声到判断推进，形成循环。

3. **边界控制是D类的难点**：题目给出的边界既是限制也是引导，需要在边界内找到推开的空间，同时不能超出边界胡编。
