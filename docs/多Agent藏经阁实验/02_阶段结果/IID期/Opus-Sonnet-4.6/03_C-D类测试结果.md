# IID期 Opus/Sonnet 4.6 C-D类测试结果

日期：2026-04-04

## C类 稀素材提炼卷

### 核心问题：会不会撑

**判断：在少料条件下完成了判断提炼与结构搭桥，未用空话硬撑。**

具体观察：

1. **判断提炼**：提炼出了一个明确的主判断——"谣言拥有一套低成本、高杀伤的传播结构"。不是泛泛说"谣言害人"，而是拆解了恐惧叠加的三层结构（病耻感 × 食物恐惧 × 熟人扩音）。
2. **结构搭桥**：12条素材中有效素材偏少，文章通过"个案现场 → 自证过程 → 反复出现的链条 → 为什么总能得手 → 围观者态度 → 结构性判断"的路径把文章撑了起来，没有出现明显的空段或注水段。
3. **硬过渡**：全文没有出现"接下来我们看看""让我们转到另一个层面"之类的机械过渡。段落之间靠叙事推进自然衔接。
4. **背景调用边界**：未编造素材池外事实。"官方反复说明""不同城市反复出现"均来自素材池原文信息。
5. **限制条件是否落住**：
   - "不能只写谣言害人"：拆解了传播结构。
   - "不能写成制度通报稿"：保留了刺痛感（孩子被排斥、帮工被劝辞、围观者建议蹭流量）。
   - "不要用空话撑长"：未出现填充段。

### 问题与不足

- "为什么这种谣言总能得手"的分析部分，三层恐惧的逻辑推进较为紧凑，但对"熟人社会扩音"这一层的具体机制还可以再挖一刀。
- 结尾指向"下一次还会再来"，判断准确但略显收束过快。

## D类 单点起稿卷

### 核心问题：会不会推开

**判断：从单点与少量锚点往外推开成功，未塌成套话扩写。**

具体观察：

1. **从单点推开**：以"你走了一条我没走成的路"这句引语为入口，从一组关系推开到三组关系，最后收到"映照"这个统一判断。推开过程有层次递进，不是平铺并列。
2. **议题判断**：选定了"差异如何成为映照而非威胁"作为主判断。三组关系分别对应"稳定vs自由""婚育vs事业""母亲vs非母亲"三种差异类型，形成递进。
3. **不胡编**：全部细节来自素材池。"核武器""一堵墙""断舍离"等表述均为素材原文引语。
4. **未平均切块**：三组关系中，第二组（婚育与事业冲突）着墨最重，写出了断联和重连的弧度。第一组和第三组篇幅较短，作为前奏和回声。
5. **限制条件是否落住**：
   - "不得写成女性励志宣言"：全文没有出现口号式表达。
   - "不得平均切块"：有主次。
   - "选定一个主判断"：映照感贯穿始终。
   - "不得用空泛抒情代替判断"：抒情段落均有具体锚点支撑。

### 问题与不足

- "映照"这个概念在文章中反复出现，最后一段的收束虽然精准，但"你的光照到了我这里"稍有散文化倾向，离特稿的叙事纪律近而离判断收束远。
- 第10条素材（补充短例）被集中压缩为一个段落处理，信息密度偏低，可以考虑进一步削减或只保留最有力的一个细节。
