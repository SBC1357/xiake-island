# B类 富素材压缩卷 成稿

---

15万块钱、一家三口、一趟从东北开出来的绿皮火车——这是一个家庭扎进南方皮毛小镇的全部本钱。那是2010年前后，小镇正处在黄金期。市场占地80多亩，挂过产业名镇的牌子，从东北、河北、河南涌来的商户把档口挤得转不开身。当时最朴素的共识只有一条：只要肯闯，就能赚钱。

钱确实赚到了。某类皮货单件卖2000元，利润约300元。做分拣整理的工人月收入能到2万。深夜还有人守在店里，等空运来的新货。过年回乡时，一个村子能看到成片挂着南方城市牌照的车。那几年的小镇，钱是能被看见、被闻到的。

然后市场放缓了。只做单一品类，越来越难撑住一家店。有的家庭转去做羽绒服，有的给电商供货，有的甚至跑到夜市摆摊。旺季仍然要两地奔波。老市场自己也在变，从只接批发，慢慢开始做零售和海外买家。但氛围不同了——不再是"干就有"，而是"得拼着找活路"。

真正有意思的变化发生在下一代。

年轻商户不像父辈那样死守档口等客人。有人把积压多年的皮毛边角料做成挂件、公仔、帽子、围巾、包和地毯，想把滞销的老材料变成新消费品。一次大促卖出了2万多件。这个数字放在以前的批发逻辑里不起眼，但它说明了一件事：消费者愿意为审美和情绪买单，前提是你得让他们觉得值。

还有人把生意搬进直播间。5个人的小团队，白天研究数据和流行趋势，晚上开播卖货。好的时候一个月出几百件，单价从几千到几万不等。后台数据也让他们意外：年轻买家不是少数。

两代人的活法被放在同一个市场里对照，差异很明显。父母一代靠迁徙、吃苦和胆量挣到了第一桶金。子女一代更擅长渠道、数据和产品重新定义。旧生意和新消费之间不是简单替代，更像被迫嫁接——老根还在，上面接的枝条完全换了品种。

那个当年坐火车南下的小孩，现在已经长大。小时候有一次想买一个10元的玩具，没买成。她没有再提过这件事。但她和她的同龄人正在做的事情，本质上是同一件：用自己的方式，在父辈的旧地基上重新开张。

小镇距杭州55公里、距上海159公里。这个位置让它在上一轮产业红利里吃到了物流和市场优势，也让下一代人在做直播和电商的时候不至于离供应链太远。地理没有变，变的是在这个坐标上讨生活的人，以及他们手里用来换钱的东西。
