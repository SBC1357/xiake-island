# C类 稀素材提炼卷 成稿

---

前一天晚上还在排队，第二天店里突然空了。

一家开了很多年的小餐饮店，几天之内客流归零。原因不是食品出了问题，也不是卫生事故，而是一句没有来路的话——老板得了艾滋病。

谣言最早大致来自微信群和熟人转述。没有人能说清楚是谁先说的，但传播链条极短：群消息转几手，一两句语音，几张截图，不需要任何证据，也不需要任何逻辑。等店主意识到发生了什么的时候，已经来不及了。

首先受冲击的是帮工。亲戚劝她们不要再去上班。孩子也被卷了进来——有人提醒别的小孩不要和他家的孩子一起玩。一句无中生有的话，在几天之内把一个家庭从正常生活里连根拔出。

为了自证清白，店主带着十几名员工一起去做了血液检测。然后在派出所、疾控中心和监管部门之间来回跑。检测结果自然没问题。但辟谣之后呢？客人不会因为一张检测报告就重新坐回那张桌子。名声这种东西，毁起来一句话就够了，修起来一年也未必够。

这类谣言并不是第一次出现。在不同城市、不同年份、不同店面里，同样的链条反复上演：被造谣、客流暴跌、报警、验血、辟谣——然后阴影不真正消失。官方早就反复说明过，艾滋病不会通过食物、餐具或饮水传播，也没有相关经食品传播的报告。这些知识写在每一份科普手册里，但拦不住一条群消息。

为什么这种谣言总能得手？

答案藏在几层恐惧的叠加里。第一层是病耻感。艾滋病在公共认知中始终挂着性道德的阴影，"得了艾滋"在很多语境里不只是生病，而是一种道德审判。第二层是对食物接触的本能恐惧——哪怕理性上知道不会传播，身体的第一反应是躲开。第三层是熟人社会的扩音效应。最可怕的往往不是一个清晰的坏人，而是"朋友的朋友""群里的消息""大家都这么说"。没有人需要为"转发"负责，每个人都只是"听说"，但每一次转发都在加固谣言的可信度。

真正追查的时候，困难也在这里。很多时候只能追到某个转发者，再往上就断了。聊天截图是证据，人情关系是阻力——你很难让一个人承认他转发的那条消息毁了别人的营生。

整件事里最让人不适的一个细节来自围观者。有直播间观众对店主说：既然事情有热度，不如顺势"接住流量"。在这个建议里，一家人的灾难被换算成了一个运营机会。伤害、自证、奔波、孩子被排斥——这些加在一起，在某些人眼里不过是一波可以蹭的热度。

问题不只是一家店垮了。问题是这种谣言拥有一套极低成本、极高杀伤的传播结构：它不需要证据，只需要恐惧；它不需要一个明确的发起者，只需要足够多的"随手转"；它击中的也不是信息盲区，而是人心里那块宁可信其有、不愿赌一口饭的本能。

每一次辟谣都只是事后止血。制造谣言的心理土壤没有被动过——病耻感还在，食物恐惧还在，熟人网络的扩音器还在，"你听说了吗"的转发成本还是零。下一次，换一家店，换一个城市，换一句措辞，同样的事情会再来一遍。

那家店后来有没有恢复，素材里没有说。但可以确定的是，店主花在证明自己"没问题"上的时间和代价，远比造谣者按下发送键的那一秒要多得多。这笔账，到现在也没有人替他们算过。
