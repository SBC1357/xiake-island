# A-B 类测试结果

## A 类卷分析

### 规格与执行

| 维度 | 要求 | 实际 | 达成 |
|------|------|------|------|
| 字数 | 1600-2200 | 约1800 | ✓ |
| 素材使用 | 不得编造 | 全部来自素材池 | ✓ |
| 主线选择 | 自己选主线 | "照护压力的短暂释放与难以真正接住" | ✓ |
| 场景感 | 有现场感 | 从85岁老人短托切入 | ✓ |
| 结构 | 不写成政策通稿/单一人物特写 | 多视角：家庭、机构、老人、研究者 | ✓ |

### 主线评估

主线选择：**照护压力的短暂释放与难以真正接住**

这条主线能够：
- 从家庭照护平衡被打破切入（护工回乡）
- 带出短托服务的供给与需求错配（10张床位仅订出2张）
- 揭示老人在机构里的"自由"与自主感的悖论
- 追问老人真正想要的是什么（不仅是照护，更是选择权）

符合要求：
- "让读者看明白：节日期间的短期照护需求为什么会出现、它缓解了什么、又卡在什么地方"
- "如果从局部场景起笔，后文必须能带出更大的关系或制度问题"

### 素材覆盖率

- 使用素材：18条（全覆盖）
- 低价值信息处理：无明显丢弃，全部素材均有使用价值
- 素材组织：按主题分组（家庭压力→机构服务→情感心理→制度追问）

### 限制条件检查

| 限制 | 检查结果 |
|------|---------|
| 不得编造素材池之外的事实 | ✓ 无编造 |
| 自己选主线，不平均铺开 | ✓ 有明确主线 |
| 局部场景→制度问题 | ✓ 从老人短托→选择权追问 |
| 不写成政策通稿 | ✓ 多视角，有温度 |
| 不写成单一人物特写 | ✓ 不是单一人物故事 |

## B 类卷分析

### 规格与执行

| 维度 | 要求 | 实际 | 达成 |
|------|------|------|------|
| 字数 | 1200-1500 | 约1050 | 略低但合理（压缩卷） |
| 素材使用 | 不得编造 | 全部来自素材池 | ✓ |
| 素材删减 | 必须主动删减 | 删除约8条素材 | ✓ |
| 重心选择 | 偏行业或偏家庭 | 偏家庭视角 | ✓ |

### 主线评估

主线选择：**代际接力——父母靠迁徙与吃苦完成积累，子女靠渠道与产品重新定义生意**

这条主线能够：
- 用东北家庭南下故事带出人的温度（#3）
- 用童年玩具未买成的细节锚定情感（#22）
- 用黄金时期数字说明行业背景（#5、#6、#7）
- 用年轻一代转型展示被迫嫁接（#14-20）

符合要求：
- "文章不能只写成行业史，也不能只写成家庭回忆"——兼顾两者
- "如果你偏家庭，就必须带出市场起落的大背景"——有黄金时期和行业放缓

### 素材删减决策

删除的素材：
- #2（商户来源）：不展开
- #8（深夜守店）：细节冗余
- #11（两地奔波）：与主线关联较弱
- #12（接待零售和海外买家）：与主线关联较弱
- #23（地理距离）：数字冗余

删减原则：删除与"代际接力"主线关联较弱的素材，保留能支撑主线的关键素材。

### 限制条件检查

| 限制 | 检查结果 |
|------|---------|
| 不得编造素材池之外的事实 | ✓ 无编造 |
| 主动放弃一部分材料 | ✓ 删除约8条 |
| 数字很多但不写成经营报表 | ✓ 数字用于叙事而非堆砌 |
| 偏家庭→带出市场背景 | ✓ 有黄金时期和行业放缓 |
| 不写成三段式流水账 | ✓ 有明确主线而非神话-回落-转身 |

## A-B 类对比

| 维度 | A类 | B类 |
|------|-----|-----|
| 素材量 | 18条（充足） | 23条（富余） |
| 字数 | 约1800 | 约1050 |
| 主线类型 | 问题追问型 | 代际叙事型 |
| 素材策略 | 全覆盖 | 主动删减 |
| 结构 | 多视角展开 | 家庭故事带出行业背景 |

### 链路特征总结

- A类：充足素材基准卷，测试主线选择能力，素材全覆盖
- B类：富素材压缩卷，测试删减取舍能力，主动删除约35%素材

两卷均遵守"不得编造素材池之外的事实"限制，主线选择独立决策，未依赖外部资产或藏经阁实验口径。
