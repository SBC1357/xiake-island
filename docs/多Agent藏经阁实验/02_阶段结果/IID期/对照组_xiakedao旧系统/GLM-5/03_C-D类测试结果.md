# C-D 类测试结果

## C 类卷分析

### 规格与执行

| 维度 | 要求 | 实际 | 达成 |
|------|------|------|------|
| 字数 | 1400-1900 | 约1600 | ✓ |
| 素材使用 | 不得编造 | 全部来自素材池 | ✓ |
| 主线提炼 | 素材偏薄，必须主动提炼主判断 | "谣言的得手机制" | ✓ |
| 文稿类型 | 个案稿或个案+机制混合 | 个案+机制混合 | ✓ |

### 主线评估

主线选择：**谣言的得手机制——为何这类谣言总能换壳重来**

这条主线能够：
- 从一家店被谣言击中切入（个案事实）
- 带出谣言为何总能得手的机制（病耻感、性道德想象、熟人社会扩音）
- 追问追查困难的原因（人情关系是阻力）
- 阐释谣言针对的是"想象"而非"事实"

符合要求：
- "既要让读者感受到一个小店和一个家庭被谣言击中的现实，也要让读者明白这种伤害为什么会反复发生"
- "你可以把文章写成个案稿，也可以写成'个案 + 机制'混合稿，但不能变成空泛控诉"——个案+机制混合

### 素材提炼策略

- 素材偏薄（12条），无法把每个面向都写满
- 选择从个案切入，用机制层面撑起文章
- 核心：#1-#6（个案事实） + #7-#11（机制层面）
- 边缘：#12（直播间围观者）作为旁注

### 刺痛感与解释力

| 要求 | 达成情况 |
|------|---------|
| 有刺痛感 | ✓ "帮工被劝退、孩子被隔离、在派出所和疾控之间来回跑" |
| 有解释力 | ✓ "病耻感、性道德想象、熟人社会扩音、人情关系是阻力" |
| 不写成空泛控诉 | ✓ 有机制分析而非只是"谣言害人" |
| 不写成制度通报稿 | ✓ 有个案叙事而非只是官方说明 |

### 限制条件检查

| 限制 | 检查结果 |
|------|---------|
| 不得编造素材池之外的事实 | ✓ 无编造 |
| 不能只写"谣言害人" | ✓ 有机制追问 |
| 不能只写成制度通报稿 | ✓ 有个案叙事 |
| 自己决定从一家店下刀还是从机制开口 | ✓ 从个案切入再往外扩 |
| 污名不能写虚 | ✓ 病耻感、性道德想象有具体阐释 |
| 不用空话把材料撑长 | ✓ 有实质内容而非空泛抒情 |

## D 类卷分析

### 规格与执行

| 维度 | 要求 | 实际 | 达成 |
|------|------|------|------|
| 字数 | 1200-1800 | 约1450 | ✓ |
| 素材使用 | 不得编造 | 全部来自素材池 | ✓ |
| 起稿方式 | 单点起稿，从核心判断往外推开 | 从#1核心判断出发 | ✓ |
| 关系深写 | 深写一组关系，其余做回声 | 深写#4、#5婚育与事业 | ✓ |

### 主线评估

主线选择：**差异作为映照——深写婚育与事业的那组，其余做回声**

这条主线能够：
- 从核心判断出发（#1）
- 深写一组关系：婚育与事业断联后重新理解（#4、#5）
- 其余两组作为回声：稳定与自由（#2、#3）、母职与非母职（#6、#7）
- 用补充短例撑开日常感（#10）

符合要求：
- "从一个点往外推开，但不能把有限素材硬拉成长篇套话"——从核心判断推开，有实质内容
- "你可以深写一组关系，其余材料做回声；但不能把所有案例平均切块"——深写#4、#5，其余做回声

### 素材推开策略

- 核心锚点：#1（核心判断）
- 深写关系：#4、#5（婚育与事业断联后重新理解）
- 回声关系：#2、#3（稳定与自由）、#6、#7（母职与非母职）
- 补充锚点：#8、#9、#10（"你走了一条我没走成的路"、身份标签前先相遇、补充短例）

### 映照感评估

| 要求 | 达成情况 |
|------|---------|
| 有映照感 | ✓ "差异不是威胁，而是映照——你走了一条我没走成的路，我很想看看你会走到哪里" |
| 不写成女性励志宣言 | ✓ 有实质叙事而非空泛宣言 |
| 不把所有人物平均切块 | ✓ 深写一组，其余做回声 |
| 不用空泛抒情代替细节和判断 | ✓ 有具体关系叙事和明确判断 |

### 限制条件检查

| 限制 | 检查结果 |
|------|---------|
| 不得编造素材池之外的事实 | ✓ 无编造 |
| 边界内推开：稳定与自由、婚育与事业、母职与非母职 | ✓ 在边界内推开 |
| 不写成女性励志宣言 | ✓ 有实质叙事而非宣言 |
| 不把所有人物平均切块 | ✓ 深写一组，其余做回声 |
| 不用空泛抒情代替细节和判断 | ✓ 有具体细节和明确判断 |
| 选定主判断，其余材料服务这个判断 | ✓ "差异作为映照"是主判断 |

## C-D 类对比

| 维度 | C类 | D类 |
|------|-----|-----|
| 素材量 | 12条（偏薄） | 10条（核心判断+少量锚点） |
| 字数 | 约1600 | 约1450 |
| 主线类型 | 机制追问型 | 关系映照型 |
| 素材策略 | 个案切入+机制撑起 | 核心判断推开+深写一组 |
| 结构 | 个案→机制→追问 | 核心判断→深写关系→回声 |

### 链路特征总结

- C类：稀素材提炼卷，测试素材偏薄条件下的主线提炼能力，个案+机制混合
- D类：单点起稿卷，测试从核心判断往外推开的能力，深写一组关系

两卷均遵守"不得编造素材池之外的事实"限制，主线选择独立决策，未依赖外部资产或藏经阁实验口径。
