## 分歧级别口径

- `高收敛`：三路在主判断上同向，只是命名或细节轻微不同，不影响当前主路线。
- `可讨论分歧`：三路主骨架仍一致，但对承重重点、条件性对象功能或边界命名有不同强调，需要压缩但不至于改写当前主路线。
- `危险分歧`：三路若按各自说法分别推进，会直接影响当前主路线、旧规则降级方式，或会把候选观察误升格成新口径。

## 跨样本总判断

1. 当前真正高收敛的，不是所有条件性对象判断，而是：
   - 旧骨架仍能解释 `8` 篇样本
   - 主拓扑大多还能被旧拓扑口径吃住
   - 主路线大多仍能落回分析型 / 人物型旧母路线
2. 当前真正危险的，不是“有没有新概念”，而是以下 `4` 类旧规则到底如何降级：
   - 评价尺子在何时会被产品导向尾段改写
   - 比喻在机制稿里到底是承重还是包装
   - 患者沟通型稿到底只是新语气，还是会改写底层动作 / 构件 / 规则
   - 监管分歧稿与阳性结果边界稿里，`CO-02 / CO-03 / CO-04` 的功能角色是否已系统漂移
3. 当前可以直接进入下一份文档处理的，不是“发明新体系”，而是把上面这些危险分歧压回旧口径里的保守写法。
