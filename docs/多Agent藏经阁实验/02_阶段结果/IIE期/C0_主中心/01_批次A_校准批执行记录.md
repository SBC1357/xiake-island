## 三、为什么是这 `8` 篇

这批不是“平均取样”，而是“校准优先”取样。

- 样本 `1` 用来看监管分歧稿能否稳定收敛到“本土评价坐标”。
- 样本 `2` 用来看多药比较稿能否稳定先立评价尺子。
- 样本 `3` 用来看“赛道重定义 + 产品唯一性”这类强主张稿会不会被过度产品化。
- 样本 `4` 用来看公司画像稿能否被稳定拆成“评价标准 -> 资产购买”。
- 样本 `5` 用来看人物稿是否真能稳定识别为“人物-问题混合”，而不是纯专访。
- 样本 `6` 用来看边界机制稿里“比喻承重 / 解释桥”是否会被稳定识别。
- supplement `7` 用来看真实项目稿中的“结果阳性 + 场景很小”边界能否被稳定抓住。
- supplement `8` 用来看患者沟通型比较稿里的“安全性降维 + 生活质量导向”是否会被稳定识别。
