## 样本 1：CGZ-YJWYJ-20240729-0022

**标题**：明朝的剑能不能斩清朝的官？论EMA和FDA对仑卡奈单抗的审评分歧  
**桶位**：审批 / 监管 / 注册博弈

### Q1 主人格是什么

**评判型分析人格**。带有明确立场倾向，敢下判断，局部带嘲讽（"反正从候诊死亡的患者数量上看，欧洲患者的抗风险能力确实是更差一些"）。不是冷站位的中立述评，而是"先亮态度再拆"的评判人格。

**结构稳定性**：稳。人格从标题（"明朝的剑能不能斩清朝的官"）到收尾（"明朝的剑也斩不了清朝的官"）全链贯穿，没有中途飘移。

### Q2 主拓扑是什么

**阶梯递进**。事件引入 → 建立评价坐标（"FDA/EMA只是本土监管机构"） → 安全性分析 → 疗效分析 → 回收到本土坐标（NMPA） → 收题。

核心结构是：**先立评价尺子（"本土机构"），再拿尺子量**。

**结构稳定性**：稳。全篇只有一条主线，没有分岔，没有并列，尺子从头用到尾。

### Q3 触发了哪些条件性对象 / 稿型子型

| 条目 | 判断 | 信心 |
|---|---|---|
| CO-02 解释桥 | 稳定触发。把FDA/EMA审评差异翻译成"本土医疗环境差异" | 高 |
| CO-03 数据锤击 | 稳定触发。ARIA发生率、CDR-SB、Centiloids数据密集使用 | 高 |
| CO-04 权威锚定 | 触发。NMPA批准+中国专家共识背书 | 中 |
| GS-03 分层递进论证 | 触发。三层：否定EMA绝对权威 → 拆安全性/疗效 → 收回本土 | 中 |

### Q4 哪些旧规则失效或需要降级

- "先立评价尺子再推进"：**成立，是主承重**。
- CO-01 比喻承重：仅马拉松比喻段局部出现，不构成主承重，此处比喻承重是**辅助级**。

无规则失效。

### Q5 这篇样本最可能暴露哪条边界

**监管分歧稿在既有分析型拓扑簇中没有专属子型**。它介于"行业分析"和"问题解释"之间，核心张力不是"解释一个机制"也不是"评估一个赛道"，而是"评判一个监管决策"。如果分析型拓扑簇继续只按"论证推进"笼统覆盖，这类"政策/监管评判"稿可能需要单独标记。

### Q6 反向调用路线

1. **先立本土评价坐标 → 拆国际争议 → 数据锤击 → 回收本土结论**（主）
2. 先陈事件 → 用解释桥翻译差异 → 回归学术/临床共识
3. 先陈两方立场 → 对比案例 → 给出作者判断

### Q7 主路线路由

**分析型主路线**——评价坐标建立 → 数据购买判断权力 → 收回本土结论。

**信心**：高

## 样本 2：CGZ-YJWYJ-20250114-0049

**标题**：终于有研究把9款已上市的EGFR-TKI拉出来比一比了……  
**桶位**：临床结果 / 机制 / 指南解读

### Q1 主人格是什么

**理性评判型分析人格**。从研究出发建立评价工具（ASCO-VF/ESMO-MCBS），用工具做排名。但后半段滑向"产品冠军叙事"——一旦伏美替尼胜出，即转入分子结构解释，评判人格让位于推广人格。

**结构稳定性**：**前稳后飘**。前半段评判人格稳定贯穿，后半段从"客观排名"滑向"单品解释推广"，人格发生偏移。

### Q2 主拓扑是什么

**阶梯递进 + 产品聚焦尾段**。评价工具建立 → ESMO-MCBS评估 → ASCO-NHB评估 → 伏美替尼突出 → 结构优化解释 → 结语。

前2/3是"先立尺子再量"的标准分析拓扑，后1/3变成"为什么冠军是它"的单品解释。

**结构稳定性**：**拓扑中途有接口断裂**。从"评比"到"解释为什么赢"的切换，不是拓扑内部的自然递进，更像两段不同拓扑的拼接。

**"评价框架+产品导向"混合稿的拓扑归属边界**。当"先立尺子"后滑入单品推广时，主拓扑从"分析型"变成"包装成分析型的产品推广稿"。暴露的是：**分析型拓扑簇下是否需要单列一个"产品导向型"子标签**。

1. **先建评价框架 → 用框架对比 → 突出目标产品 → 结构优化解释 → 收**（主）

**"人物-问题混合"的比例边界**。当药物解释占比达到40%时，"人物稿"标签还能不能稳定贴？暴露的是：**人物-问题混合稿需要一个比例阈值**——超过某个点，它就不再是"人物稿加了问题解释"，而是"问题解释稿加了人物入口"。本篇尚在人物侧，但已接近临界。

## 样本 7：SUPP-PROJ-001

**标题**：FLAURA2最终OS结果深度解析，联合化疗的优势很大，但场景很小……  
**桶位**：临床结果 / 机制 / 指南解读（supplement）

### Q1 主人格是什么

**审慎评判型人格**。不是简单好/坏判断，而是高颗粒度拆解"谁获益、谁不获益、有没有替代方案"。带批判性视角：承认OS阳性→但拆获益来源→限制适用范围→给出替代方案。

**结构稳定性**：稳。审慎评判人格从"优势很大但场景很小"的标题到"量体裁衣"的收尾全链贯穿。

### Q2 主拓扑是什么

**阶梯递进+条件性分流**。三大节顺序递进（获益来源→获益人群→替代方案），但第二、三节内部是条件性分流：按亚组拆人群，按场景拆方案。

核心结构是：**先承认总体结果 → 拆真实获益来源 → 划定真正获益人群 → 提出替代方案 → 精准分流结论**。

**结构稳定性**：稳。三节递进节奏清晰，内部分流逻辑自洽。但后半段在推荐伏美替尼替代方案时，产品推广色彩再次出现。

### Q3 触发了哪些条件性对象 / 稿型子型

| 条目 | 判断 | 信心 |
|---|---|---|
| CO-03 数据锤击 | 强触发。OS/PFS/HR/ORR数据密集 | 高 |
| CO-02 解释桥 | 强触发。研究设计/统计学→临床决策语言的翻译 | 高 |
| CO-05 风险解剖 | 局部触发。拆联合化疗的毒性/依从性风险 | 中 |

### Q4 哪些旧规则失效或需要降级

- "先立评价尺子"：**成立**。先明确OS获益真实来源，再做人群分流，尺子是"获益来源"。
- "算账推演"：**成立**。逐亚组拆"谁受益谁不受益"。
- **需要标记**："条件性分流拓扑"在既有拓扑簇中没有专属位置。当前分析型拓扑簇覆盖了"阶梯推进""双层递进""并列推进"，但没有"条件性分流"——即"同一个结论对不同人群给出不同路由"的拓扑。

### Q5 这篇样本最可能暴露哪条边界

**"条件性推荐"拓扑的边界**。当文章核心论点是"限制适用范围"时，拓扑不是典型的"正面推荐"也不是"否定"，而是"哪些人用A方案、哪些人用B方案"的条件性分流。暴露的是：**分析型拓扑簇是否需要增加"条件性分流"子型**。

### Q6 反向调用路线

1. **先承认结果阳性 → 拆获益来源 → 划分获益人群 → 提出替代方案 → 精准分流**（主）
2. 从患者视角 → 联合化疗意味着什么 → 谁适合谁不适合
3. 从临床决策全景 → 所有强化治疗方案比较 → 精准匹配
4. 从FLAURA2局限 → 反推精准治疗需求

### Q7 主路线路由

**分析型主路线——条件性评判+精准分流**。

**信心**：高

**桶位**：疾病 / 赛道 / 治疗格局综述（supplement，患者沟通型）

**但需要标记**：**患者沟通型人格在当前人格体系中没有位置**。既有人格体系的讨论（IIA/IIB）一直围绕"评论型""叙事型""分析型""人物型"等专业写作人格，从未覆盖过"患者教育/科普"人格。

### Q2 主拓扑是什么

**对比推进+单边推荐**。方案盘点 → 药物背景科普 → 安全性对比 → 疗效对比 → 推荐伏美替尼。

表面是"对比"，实际是"走向已定的单边推荐"——从始至终推荐答案已经明确。

**结构稳定性**：**拓扑自洽但有"伪对比"特征**。形式是对比，实质是推荐。所有数据组织都指向同一个结论。

### Q4 哪些旧规则失效或需要降级

- "先立评价尺子"：**不成立**。这篇不是先立尺子再量，而是"先给结论再找数据支撑"。**"评价尺子→全员评估"规则在患者沟通型稿件中失效**。

**人格维度的类型边界**。当前人格体系假设人格是"上层组合映射"——即底层四层不变，人格只是选择不同的组合偏好。但患者沟通型人格不只选择不同偏好，它**改变了底层运作方式**（降维科普替代精确解释、共情替代权威锚定、收尾答案预设替代开放评判）。暴露的是：**人格上收的框架是否足以覆盖"改变底层运作方式"的人格类型**。
