### 样本16：`CGZ-YSD-20210610-0005`

**标题**：从Aducanumab到GV-971，人类能跳出阿尔兹海默症这座"五指山"吗？

#### 1. 主人格是什么

**历史回望型赛道解读者**

文章以《西游记》典故开篇，把阿尔兹海默症比作"五指山"，把 FDA 的审批决定比作"唐僧手"，全程带着历史纵深感和叙事口吻审视 AD 药物研发困局。人格不是单纯的数据分析者，而是能借用文化典故做框架映射的解释者。

#### 2. 主拓扑是什么

**双线并列拓扑**

- A 线：FDA 审批争议（Aducanumab 正反方观点、王亚宁博士解读）
- B 线：中国 GV-971 对标（两年前的审批争议、数据争议、科学性争议）
- 收束：把两线并置，映射到同一"困局"框架

拓扑形态是 `历史比喻起手 → FDA 审批争议拆解 → 中国对标 → 问题收束`。

#### 3. 触发了哪些条件性对象 / 稿型子型

| 条目 | 是否触发 | 触发证据 |
|------|----------|----------|
| CO-01 比喻承重 | **是，主承重** | "五指山"比喻承重全文组织骨架，不是装饰。开头引用《西游记》原文，全文围绕"困局"和"能否跳出"展开，比喻定义了文章框架 |
| CO-02 解释桥 | 是 | 需要解释 Aβ 假说、FDA 专家委员会争议、定量药理学决策逻辑 |
| CO-04 权威锚定 | 是 | FDA 历史先例（反应停）、王亚宁博士观点、患者组织声音 |
| CO-07 分析型拓扑簇 | **弱** | 虽有分析成分，但主推进靠叙事和比喻，不是典型分析型 |

**稿型子型**：无明显触发。GS-06 意象回归收尾有弱痕迹（开头的"五指山"在结尾被呼应），但不是主承重。

#### 4. 哪些旧规则失效或需要降级

| 旧规则 | 状态 | 说明 |
|--------|------|------|
| 比喻只做修辞点缀 | **失效** | "五指山"不是机制比喻，而是困局比喻，承重全文骨架 |
| 赛道稿与监管稿互斥 | **需要降级** | 本篇把 FDA 审批争议、中国对标和赛道前景同时写入一个拓扑里 |
| 分析稿以数据为主承重 | **需要降级** | 本篇数据不多，主承重是历史叙事和比喻框架 |

#### 5. 这篇样本最可能暴露哪条边界

**比喻承重在赛道稿中的边界**

- 批次 A 样本 6 的"脑机接口门诊的一个下午"触发的是机制比喻（用具体场景解释抽象机制）
- 本篇触发的是困局比喻（用文化典故框架化整个赛道困境）

这两类比喻功能不同：
- 机制比喻：降低理解门槛，解释"它是怎么工作的"
- 困局比喻：框架化问题空间，回答"我们被困在哪里"

**边界暴露**：CO-01 的触发条件需区分"机制比喻"与"困局/问题框架比喻"，当前口径只覆盖前者。

#### 6. 如果要反向调用，应给出哪 1-5 条写法路线

1. 困局比喻型赛道综述稿
2. 审批争议 + 中国对标稿
3. 历史回望型疾病赛道稿
4. 正反方辩论型解读稿

#### 7. 最终你会把它路由到哪一条主路线

**归入分析型母路线里的赛道综述子型**

但需标记边界暴露：比喻承重在此属困局框架而非机制框架。CO-01 当前口径未覆盖"困局比喻"子类。
