## 交叉裁决规则

1. `C1` 判"观察项，暂不升格" → 默认 `保留为观察项，不进入本轮复验主单元表`
2. `C1` 判"旧容器"或"有限组合" → 即使 `C2` 给了明确功能归属，也 `不得按单条进入复验，只能按组合处理`
3. `C1` 判"最小单元，可保留" + `C2` 给明确功能归属（写作目的候选 / 条件性对象候选 / 稿型子型候选） → `可作为最小单元进入复验`
4. `C1` 判"最小单元，可保留" + `C2` 判"旧母路线容器" → 结构可独立但功能定位在架构容器层，不构成可测试的写作单元 → `保留为观察项，不进入本轮复验主单元表`
