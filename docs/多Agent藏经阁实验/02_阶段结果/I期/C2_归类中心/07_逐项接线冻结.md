## 当前公式主链

```
公式1: 写作目标 * 排列规则 = 写作目的
公式2: 写作目的 * 人格 = 写作范围
公式3: 写作范围 * 最小内容单元 = 最小有效内容单元
公式4: 写作范围 * 最小逻辑单元 = 最小有效逻辑单元
公式5: 最小有效内容单元 * 内容单元组合规则 = 内容组合
公式6: 最小有效逻辑单元 * 逻辑组合规则 = 组合逻辑
公式7: 内容组合 * 组合逻辑 = 文章大纲
公式8: 写作范围 * 文章大纲 = 有效大纲
```
