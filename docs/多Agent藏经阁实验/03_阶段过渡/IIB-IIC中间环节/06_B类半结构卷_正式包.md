阶段：IIB -> IIC 中间环节  
用途：`IIC` 第一轮 `B类半结构卷` 正式包

说明：本文件已做去身份处理，不提供作者名、刊物名、原标题和原文顺序。
定位：这是中等提示强度试卷。会给你事实、时间信息和少量人物线索，但不会替你总结矛盾，也不会提前写出主问题答案。

## 一、任务要求

请基于下面的素材，写一篇中文深度稿。

最低要求：

1. 不得编造素材之外的事实。
2. 文章不能只写成行业史，也不能只写成家庭回忆。
3. 文章需要同时处理一座市场的兴衰、一个迁徙家庭的应对，以及新旧生意逻辑的更替。
4. 允许舍弃材料，但必须保留能支撑完整判断的主线。

建议输出：

- 字数：`1800-2600` 字
- 面向：普通中文读者
- 目标：让读者看到，一门看似老旧的生意，如何在二十年里从淘金神话走到艰难转身

## 二、素材

### 1. 起点材料

1. 东部一座小镇的皮毛市场，占地 `80+` 亩，曾被正式授予过某种“产业名镇”称号。
2. 这个市场最鼎盛的时候，吸引了来自北方多个省份的人集体南下。
3. 有一个家庭在孩子很小时就从东北坐绿皮火车南下，带着东拼西凑的 `15` 万元开始在这里扎根。
4. 当时的共识很简单：只要肯闯，就能赚钱。

### 2. 鼎盛期材料

1. 大约在 `2010` 年前后，这里处于黄金时期。
2. 某类皮货单件售价可到 `2000` 元左右，单件利润约 `300` 元。
3. 做分拣或整理的工人，一个月收入可达 `2` 万元。
4. 即使深夜，店里也有人守着，等空运来的新货。
5. 那些年，返乡时一个村里都能看到成片来自南方城市牌照的车。

### 3. 变化材料

1. 行业后来明显放缓，原来只做单一品类的方式越来越难撑住一家店。
2. 有的家庭开始转去做羽绒服、电商供货，甚至夜市摆摊，但一到旺季又不得不两地奔波。
3. 这个老市场也在变：从只做批发，慢慢开始接待零售和海外买家。
4. 商户们把皮草与牛仔、针织、羽绒等不同面料混搭，试图做出更轻、更年轻、更容易成交的东西。

### 4. 新路子材料

1. 有年轻商户把积压材料做成小挂件、公仔、帽子、围巾、包、地毯等周边，试图把老材料变成新消费品。
2. 他不一定是最早想到这条路的人，但却是少数坚持做下来的人。
3. 他把生产和加工放在北方老家，把研发和销售留在市场所在的小镇，形成了两地协作。
4. 某次大促期间，这类小件卖出了 `2万+` 件。
5. 他的判断是：当基本生活水平提高之后，消费者不再只满足“穿暖”，还会为审美、趣味和情绪价值买单。

### 5. 另一条新路子

1. 还有年轻人把这门生意搬进直播间，组起一个很小的团队。
2. 他们开始研究数据、猜测下一年流行的是哪种材质、哪种款式、哪种长度。
3. 单件产品价格带从几千元到几万元不等。
4. 生意好的时候，一个月能卖出几百件。
5. 从后台看，年轻消费者并不是少数。

### 6. 人物与关系线索

1. `父母一代`：靠迁徙、吃苦和胆量进入市场，在景气期完成原始积累。
2. `子女一代`：比上一代更擅长渠道、周边、直播、数据和产品重新定义。
3. `市场` 与 `家庭`：彼此塑造。家庭命运随着市场起落被反复改写。
4. `旧生意` 与 `新消费`：不是简单替代，更像是被迫嫁接。

### 7. 引语材料

1. 有人回忆那个年代时说：只要敢闯，就能挣到钱。
2. 有人觉得，如果只守着单一产品，已经很难支撑一家企业。
3. 有人把自己做的小周边畅销，理解成消费需求的变化，而不只是应急求生。
4. 有摊主卖货时最常说的一句话很朴素，大意就是：喜欢就带一件。

### 8. 噪音与次要材料

下面这些信息未必是主承重：

1. 小时候想买一个 `10` 元玩具但没买成。
2. 夜市摆摊时，一天运气好能赚上千元。
3. 市场春节期间只有少数店铺开门，因为商户大多返乡。
4. 某些小件的具体款式和颜色。

## 三、执行提醒

1. 你需要自己决定，这篇文章的重心到底是“行业变了”，还是“人怎么跟着变”。
2. 如果你只写怀旧，会变轻；如果你只写电商和市场变化，会变硬。
3. 数字很多，但不要把文章写成一串经营报表。
4. 你需要自己组织“神话、回落、转身”之间的关系。
