## 一、任务要求

请基于下面的素材碎片，写一篇中文深度稿。

最低要求：

1. 不得编造素材之外的事实。
2. 你需要自己判断这篇文章到底是个案稿、社会问题稿，还是“个案 + 机制”混合稿。
3. 文章必须既能让读者感受到一个小店和一个家庭被谣言击中的现实，也能让读者明白这种伤害为什么会反复发生。
4. 允许删减碎片，但删减后仍要保留足够的社会解释力。

建议输出：

- 字数：`2200-3200` 字
- 面向：普通中文读者
- 目标：写出一种谣言如何在熟人社会、平台传播和病耻感中完成闭环

## 二、素材碎片池

下面这些材料都是有效输入，但不保证都重要，也不保证顺序。

1. 一家开了很多年的餐饮小店，几天之内突然“没人来了”。
2. 原因不是食品出问题，也不是卫生事故，而是一句没有来路的话：老板得了艾滋病。
3. 这类针对餐饮店的谣言，近几年在不同城市反复出现，几乎已经形成固定链条：被造谣、客流暴跌、店主报警、验血、通报、辟谣，但阴影并不真正消失。
4. 这家店开在小城巷子里，却是本地热门榜上的第一名，只做堂食，每天营业时间并不长，原本常常排队。
5. 店主大约 `35` 岁，做餐饮十几年，把一间小铺一点点扩成一百多平方米。
6. 谣言最早似乎来自微信群和朋友转述，版本大意是：店主赚到钱后，在外面“乱来”。
7. 消息先在熟人之间传，再扩到菜市场、街巷、家属和孩子同学的家长那里。
8. 前一晚还在排队，第二天就突然空了。
9. 帮工也受到波及，亲戚劝她们不要再来上班。
10. 有人打了政务热线举报，说店主明知自己患病还在继续经营。
11. 市场监管部门来查过手续，也找店主谈话，事情一下子升级了。
12. 为了证明自己没问题，店主和十几名员工一起做了血液检测。
13. 这时他才意识到，一句起初荒唐到不值得理会的话，已经变成一件足以改变生活的大事。
14. 更残酷的是，孩子也会被卷进来。有人会叮嘱别的小孩不要跟他的孩子一起玩。
15. 被造谣的人自己，也会在不知不觉中把外界的羞耻感内化，觉得别人都在用异样眼光看自己。
16. 官方早就反复说明，艾滋病不会通过餐具、饮水或食物传播，历史上也没有相关经食品传播的报告。
17. 但从“针”“水果”“牙签”“烧烤”到“餐饮店老板”，谣言总会不断换壳。
18. 另一地另一个餐饮家庭，也遭遇过几乎同款谣言。他们最后把检测结果贴在店门口，也发到网上。
19. 还有别的店，因为同类谣言，连续二十多天没有生意，营业额几乎归零。
20. 有人愿意拿钱悬赏找线索，也有人在直播间辟谣，希望把生意拉回来。
21. 但辟谣这件事特别耗人：要去派出所、疾控、各个部门来回跑，要想办法收集聊天截图，还要面对熟人社会里“不好意思把朋友供出来”的顾虑。
22. 在一起案例里，最后追到的并不是源头，只是某个转发者。对方说自己也是听别人说的。
23. 即使找到这个人，也未必走向彻底追责。人情、孩子、熟人关系，都会把后续处理重新拖回暧昧地带。
24. 有人道歉、写保证、拍视频，但那不等于生意马上恢复。
25. 还有人冷冷地提醒：既然事情已经有热度，不如顺势“接住流量”。
26. 这种说法进一步放大了受害者的荒诞感：明明在努力证明自己没有病，却被要求把伤害重新包装成经营机会。
27. 从社会机制上看，这类谣言能反复得手，离不开病耻感、性道德想象、对食品接触的恐惧，以及小城熟人网络的扩音。
28. 从传播结构上看，最可怕的不是一个明确坏人，而是“朋友的朋友”“群里的消息”“大家都这么说”。
29. 被谣言击中的不是抽象个体，而是整个家庭、店铺名声、员工生计和孩子的学校生活。
30. 就算得到官方正名，很多人也不会专门回来告诉你：我之前误会了。

## 三、噪音与次要材料

下面这些信息可以视为噪音或次要材料：

1. 某些店具体卖的是哪种地方小吃。
2. 个别年份、月份和外地案例的精确时间点。
3. 某些截图、视频、道歉文本的表述细节。
4. 直播平台里围观者的情绪化发言。

## 四、执行提醒

1. 这篇不能只写“谣言害人”，那样太浅。
2. 也不能只写成制度通报稿，那个会失去人的刺痛感。
3. 你需要自己决定，是从一家店下刀，再向外扩；还是从机制开口，再落回一家店。
4. 如果你把“污名”这件事写虚了，会直接失真。
