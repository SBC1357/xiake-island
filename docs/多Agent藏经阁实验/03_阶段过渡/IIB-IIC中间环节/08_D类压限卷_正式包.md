# D类压限卷_正式包

状态：Frozen  
日期：2026-04-04  
阶段：IIB -> IIC 中间环节  
用途：`IIC` 第一轮 `D类压限卷` 正式包

说明：本文件已做去身份处理，不提供作者名、刊物名、原标题和原文顺序。
定位：这是第一轮最难的卷。材料处理强度不低于 `C类`，同时额外加入有限写法约束，用来测试执行中心能否在不靠作者标签的前提下，写出接近原稿家族的作品。

## 一、任务要求

请基于下面的素材碎片，写一篇中文特稿。

最低要求：

1. 不得编造素材之外的事实。
2. 这不是单一人物传记，也不是议论文。
3. 文章需要让读者感受到：不同人生路径上的女性，如何在彼此身上获得理解、底气与照亮。
4. 你可以保留多组故事，也可以以其中一组为主、其余作为回响，但不能把所有人物平均分配成流水账。
5. 必须主动识别并丢弃营销或口号型噪音。

建议输出：

- 字数：`2200-3200` 字
- 面向：普通中文读者
- 目标：写出“差异并不削弱连结，反而让连结更有力量”的关系特稿

## 二、素材碎片池

1. 一组材料围绕几对女性朋友展开。她们的人生走向很不一样：有人追求稳定、责任与秩序；有人持续漂泊、试错和冒险；有人进入婚育与家庭；有人在事业里向前冲；有人有孩子，有人没有。
2. 第一组里，一位女性在大城市的高压职场里一路向前，长期把“稳定”视为目标；她最好的朋友则很早离开固定上班轨道，去海外做过打工、旅居和自由职业相关的事。
3. 这对朋友重新见面时，一个问：到底什么时候才能财务自由、不用上班？另一个反问：那你觉得多少才算够？
4. 这个反问后来变成了一次认知松动：原来很多焦虑并没有答案，只是被外部标准推着走。
5. 那位更“稳定”的人，后来把朋友看作一种灯塔；而那位更漂泊的人，则把这段友谊看作自己不必断舍离的安全网。
6. 她们有过一通非常长的视频电话。事后双方才知道，当时两个人都以为是“对方更需要陪伴”，所以谁也没有先挂。
7. 第二组里，两位从小地方一起长大的女性，原本一直是彼此最紧的人。
8. 后来其中一人经历婚育、难产和身体创伤，不得不暂时放下原本的事业；另一人则继续在事业上加速向前。
9. 最痛的一次误会来自一个时刻：一个人在产后和混乱里需要朋友，另一个却在公众平台上像是正要进入新的人生航线。
10. 前者因此把后者拉黑，觉得自己在最需要的时候被丢下了。
11. 但很多时间过去之后，她们重新联系，才发现彼此都在各自那条路上吃尽了苦头。
12. 那位被想象成“事业大女主”的人，外表高光，背后却像陀螺一样转，身体和情绪都付出过很大代价。
13. 那位进入家庭的人，也不是外界想象的“圆满归位”，她真实经历的是停滞感、价值感下降和自我怀疑。
14. 她们后来共同承认：所谓“妈妈”“事业女性”这些标签，并不能解释她们真正成为了谁。
15. 第三组里，两位年龄更大的女性，一个在育儿、养老和日常焦虑里穿梭，一个没有走向母职，却把生活过得很有玩心。
16. 她们会利用午休一起吃饭、聊天、到处玩，也会在对方人生最重的时候，陪着去医院、等检查、接住情绪。
17. 其中一位说过一句极有代表性的判断：和朋友“扯闲篇”的治愈力，强到像某种重型武器。
18. 她们并不以身份先认识对方，不是“丁克对妈妈”，而是“一个人对另一个人”。
19. 这些关系里反复出现的，不是相同，而是差异被理解、被尊重、被接住。
20. 文末还补入了更多简短例子：不同城市、不同阶层、不同生活轨道的女性，都在讲相似的经验。
21. 其中有人会说：你走了一条我没走成的路，我很想看看你会走到哪里。
22. 也有人会说：你事业再成功，想找个地方吐槽时，我会在。
23. 整篇材料试图把“不同”从潜在隔阂，写成一种彼此映照与互相成全的来源。

## 三、附加限制

这一卷除了素材，还附加以下写法限制：

1. `人格`
   - `温柔说书人`
2. `主拓扑`
   - `多组镜像人物串联`
   - 允许一组深写、两组辅助回声，但全文必须回到同一个关系判断
3. `情绪温度`
   - `中高`
   - 要有温度，但不能过热，不能喊口号
4. `证据密度`
   - `中`
   - 主要靠细节买权，不靠抽象论断硬顶
5. `收尾方式`
   - 从具体关系回收到更普遍的经验判断，再轻轻落回人与人之间的照亮感
6. `禁用项`
   - 不得直接写品牌、产品、征集活动
   - 不得写成“女性励志宣言”
   - 不得把所有人物平均切块
   - 不得用大段空泛抒情代替细节

## 四、噪音与干扰项

下面这些材料可以视为噪音或次要材料：

1. 某些故事后面附带了更广泛的征集背景。
2. 某些段落带有明显的品牌式价值口号。
3. 有些案例只是补充举例，不一定值得占据正文篇幅。
4. 个别具体物品、消费和产品信息，对主判断帮助很小。

## 五、执行提醒

1. 你需要自己决定，是把文章建在“一组主故事 + 两组回响”上，还是建在“三组并列镜像”上。
2. 如果你把它写成鸡汤，会直接失败。
3. 如果你为了“相似度”而照搬显性口号，也会失败。
4. 真正要逼近的不是句子表面，而是那种“温柔、具体、彼此映照”的结构家族。
