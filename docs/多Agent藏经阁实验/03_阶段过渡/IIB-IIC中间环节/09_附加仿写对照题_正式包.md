# 附加仿写对照题_正式包

状态：Frozen  
日期：2026-04-04  
阶段：IIB -> IIC 中间环节  
用途：`IIC` 第一轮附加仿写对照题正式包

说明：本文件已做去身份处理，不提供作者名、刊物名、原标题和原文顺序。  
定位：这不是 `IIC` 主门槛的一部分，而是一道附加控制题。它专门测试：面对一篇已验证过的人物向旧 `B` 稿家族，执行中心能否在闭卷条件下，写出足够接近的人物核心轴与材料回钉结构，而不是只会写成普通企业人物稿。

## 一、任务要求

请基于下面的素材，写一篇中文人物特稿。

最低要求：

1. 不得编造素材之外的事实。
2. 这不是创业公司宣传稿，也不是纯行业解释稿。
3. 文章必须让读者清楚感到：一个人对“不合理”的强烈反应，以及他对“确定性”的偏执，如何一路控制了他的判断、动作和选材。
4. 行业、产品、供应链、环保等解释段落，必须反复回到这个人本身，不能各写各的。
5. 允许舍弃具体功能细节，但不能把人物核心轴写散。

建议输出：

- 字数：`2200-3200` 字
- 面向：普通中文读者
- 目标：写出一篇“人物推动问题、问题反证人物”的特稿，而不是一篇整齐但无骨的商业人物稿

## 二、素材

### 1. 起手材料

1. 一位曾在新能源行业做到高位的人，后来转去做“居住空间整体产品”。
2. 在他看来，人们今天能在车里享受前沿科技，却还要在家里忍受高度落后的体验，这之间存在巨大的时代错位。
3. 他把传统家装形容成一种“攒机式”模式：不同品牌、不同设备、不同渠道拼到一起，既不匹配，也有大量不透明成本。
4. 正是这种错位和不满，把他重新推到了一号位。

### 2. 人物核心轴材料

1. 这个人几乎对什么都不满意。
2. 他的说话习惯是，指出问题时会带着很强的否定力度，近乎形成口头禅。
3. 他很容易被“伪科技”“体验割裂”“结构不合理”这种事激怒。
4. 他不是情绪化抱怨，而是会立刻追问：为什么会这样，能不能彻底改掉，怎样才能把结果做成确定的。
5. 他精力极旺盛，开完长会仍可能继续兴奋地讲细节、指现场、拆问题。

### 3. 过往经历材料

1. 他早年在外企做高管时，已经很强烈地感受过时代周期带来的不确定。
2. 那种不确定感，后来驱使他跳去更有发展前景的新赛道。
3. 在新能源汽车行业的多年经历，让他第一次把“确定性”做成了结果。
4. 但这种对确定性的渴求没有减弱，反而更强。
5. `2022` 年前后，一次腿伤把他按在家里，也让他第一次长时间正面感受到装修和居住体验里的大量荒诞。

### 4. 行业判断材料

1. 他算过一笔账：这个大行业规模极大，但传统渠道、租金、获客和返点吃掉了大量成本。
2. 一个普通家庭花几十万元装修，真正落在自己身上的那部分，远低于表面账单。
3. 传统交付更像原材料和零散设备的堆砌，不是一个完整产品。
4. 在这种模式下，所谓“智能”常常只是功能割裂、设备各管各的假智能。
5. 他的核心判断是：用户买的不是零件，而应该是确定的结果。

### 5. 解决方案材料

1. 他的定义不是“卖若干智能设备”，而是直接交付一个软硬件一体化的空间产品。
2. 在他的内部排序里，优先级非常死板：先健康，再舒适，再智能。
3. 他不相信一堆花哨功能能掩盖底线问题；如果最基本的健康和可用性做不到，别的都是空话。
4. 他反复强调：老人会不会用，孩子会不会用，自己家愿不愿意装，是所有方案都得过的三道关。

### 6. 睡眠场景与传感材料

1. 他拿过一组卧室实测数据：密闭睡眠环境下，某项空气指标在半夜会升到远高于健康底线的水平。
2. 他由此认为，传统家装根本没有为“最终健康结果”负责。
3. 他的解法不是再叠加单台设备，而是让传感器、算法、通风、温控协同工作。
4. 在他描绘的理想场景里，入睡、深睡、换气、唤醒都应该尽量无感完成。
5. 这也是他眼里“真智能”和“伪科技”的分水岭。

### 7. 空间与功能材料

1. 他对空间利用也极度敏感，认为很多家庭真正被浪费的不是面积，而是体积。
2. 他反对无功能的“木头盒子”，强调功能密度。
3. 为了把不同系统塞进一个更顺手、更省空间的结构里，他会逼着团队、供应商和工艺一起改。
4. 某些模块为了兼顾功能和空间，甚至做出多个变体版本。
5. 在他看来，美不是贴上去的装饰，而是结构与功能被打通后自然长出来的秩序。

### 8. 环保与实验室材料

1. 他最不能妥协的一块，是环保。
2. 他认为行业里很多供应商只敢保证单一材料，却不对最终叠加结果负责。
3. 因此他要求的是：所有材料最终叠加到真实空间里，结果依然达标，甚至要为用户后续自购物品预留余量。
4. 为了这件事，公司早期就砸钱搭建高规格实验室，做大量交叉测试。
5. 测试里被淘汰的材料远多于能留下的材料。
6. 他甚至停掉过利润不错的配套业务，只因为量产结果和送测样品不一致。
7. 他说自己不相信运气，只相信算出来、控出来的确定性。

### 9. 一号位与时代判断材料

1. 成为一号位之后，他更累，但也更兴奋。
2. 对普通人来说，这种长期高压可能难以承受；对他来说，却接近某种上瘾的刺激。
3. 与他对产品和标准的强势形成对比的是，他并不把自己理解成改变时代的英雄。
4. 他承认个体常常只是踩中了时代窗口。
5. 但他仍然愿意把未来十年甚至更长时间，押在“把居住体验真正推进一代”这件事上。

## 三、附加限制

这道附加题除了素材，还附加以下写法限制：

1. `人格`
   - `冷静分析师`
   - 但人物核心轴必须强显性，不能冷到只剩行业判断
2. `主拓扑`
   - `单人物核心轴 + 多轮问题回钉`
   - 行业、产品、环保、空间等材料都要成为人物轴的反证，不得平铺并列
3. `情绪温度`
   - `中低`
   - 要有硬度和判断力，但不要写成冷冰冰的说明书
4. `证据密度`
   - `高`
   - 需要靠场景、数据和细节持续买权
5. `收尾方式`
   - 从人物当下的强迫式控制，收回到一个更长时间尺度上的行业愿景，但不能喊口号飞起来
6. `禁用项`
   - 不得写成企业宣传文
   - 不得只堆产品功能
   - 不得把人物和问题拆成两篇稿
   - 不得用空泛热血替代细节

## 四、噪音与次要材料

下面这些信息可以视为噪音或弱承重信息：

1. 某些产品线名称和型号名
2. 某些具体空间命名和营销式包装说法
3. 某些算力、模块数量、版本数量等参数
4. 某些“时代已经来了”的大口号

## 五、执行提醒

1. 这道题不看你会不会模仿句子表面，重点看你能不能把“人物核心轴”重新立住。
2. 如果你把它写成普通行业分析稿，会直接失败。
3. 如果你把行业判断写得很多，但人物只剩几个标签，也会失败。
4. 真正要逼近的，不是某个金句，而是那种“一个人的不满与确定性，如何反复把所有材料吸回自己身上”的结构家族。
