## 阶段决议

1. `IIE` 提前终止。
   - 终止原因不是旧骨架已被直接证伪，而是发现：`IIE` 之前形成的一批“说法”，很可能混入了大量非最小单元的有限组合。
   - 若继续在 `IIE` 内压边界、压条目、压表，容易把组合态误冻成正式规则。
2. `IIE` 结果区全部保留，但只作为“重大隐患触发证据”使用。
3. 正式开启 `IIF`。
   - `IIF` 的唯一目标：把 `IIE` 之前所有发现过的说法全部拎出来，做一次最小单元审计，然后用 `IIE` 的 `32` 篇稿子再验证一轮。
4. `II期` 是否整体失败，不在本轮直接宣判。
   - 该裁决延后到 `IIF` 结束后再做。
5. 面向桥接层、生成层和侠客岛反推层的讨论全部后置。
   - `IIF` 未跑完前，不再把桥接层当成当前主线任务。
