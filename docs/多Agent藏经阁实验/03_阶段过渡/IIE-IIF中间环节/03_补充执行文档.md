## 二、`IIF` 当前角色划分

1. `C0_审计整编中心`
   - 负责把 `IIE` 之前所有发现过的说法全部整编出来

2. `C1_结构裁判中心`
   - 负责判断每条说法是不是最小单元，还是组合态、旧容器或案例表达
   - 重点看结构可拆性与是否混入多个动作 / 多个承重判断
3. `C2_功能裁判中心`
   - 负责判断每条说法的功能归属更像什么
   - 重点看它更像 `写作目的候选 / 条件性对象候选 / 稿型子型候选 / 旧母路线容器 / 暂不能判`
