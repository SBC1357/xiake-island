## 多 Agent 实验工程方法论

状态：当前实验中心固定工程规则  
适用：`IID` 及后续同类实验段的方案修订、挑战、前置工程与平行执行  
性质：这是工程组织方法，不是实验阳性结论

### 1. 主 session 职责固定

1. 只负责明确本轮唯一实验目标，不把目标外包给执行 Agent。
2. 负责冻结主命题、阶段口径、变量边界、失效条件与裁决口径。
3. 负责把可并发部分拆成互不阻塞的子任务，并指定角色。
4. 负责接收流式回报；发现一个阻塞项，就立即裁决、修订并回写正式文件。
5. 负责最终阶段判断；子 Agent 只能交证据、问题和修订，不得自行宣称阶段完成、进入人工终审或进入下一期。

### 2. 并发角色固定

1. 方案撰写与完善：把主 session 冻结下来的目标压成可运行实验方案与正式口径。
2. 方案挑战：专门找可证伪点、变量污染、越级结论、口径打架与证据缺口。
3. 前置工程补全：补齐实验开跑前必须先具备的人工、半人工或临时可用能力。
4. 前置工程系统化配置：把已可用的工程样板压成脚本、配置、入口或可复用流程。
5. 其他可并发操作的执行 Agent：只承担不阻塞主裁决的平行执行项，例如素材整理、目录映射、样板验证、单题回填或其他局部执行。

### 3. 并发执行规则固定

1. 能并发就并发，但只拆不共享同一写面、且不互相阻塞的任务。
2. 子线必须流式交付；一个样板、一个证据包、一个阻塞项，都应先上报，不等整包。
3. 方案挑战线发现一个问题，就应立即触发主 session 或修订线即时修订，不允许攒一批再报。
4. 前置工程线只要有一条先交出可用样板并被验证，对应实验启动阻塞即可解除；另一条可以继续慢跑做系统化配置。
5. 主 session 可以按当前证据假设某条前置工程线已满足最低闸门，并继续推进方案固化；但必须把该假设和已验证边界写进正式文件，不得偷换成“全部已完成”。

### 3.5 子线派工硬约束

1. 主 session 已完成前置阅读回执后，子 session 不得重复补读同一批仓库前置文件，除非本轮派工明确要求它处理新的正式文档。
2. 当前 Codex 嵌入式 PowerShell 环境下，派工必须直接写明：不要用 `rg`，也不要再探 `python / pandoc`；检索一律用 `Get-ChildItem / Select-String / Get-Content`。
3. 子 session 不得把环境探测、工具查找或“先找解释器”当成正式进度回报；回报只认新增证据、阻塞项或最终交付。
4. 只要任务涉及题目级资料卡，派工必须同时写明两个落点：
   - 仓外 runtime 目录
   - `D:\汇度编辑部1\项目文章\侠客岛测试任务\<对应题目文件夹>\g0_evidence_card.md/json`
5. `D:\汇度编辑部1\项目文章\侠客岛测试任务` 从本条开始固定为后续所有写作系统优化的完整题库根目录；根目录只保留索引，资料卡必须回填到各自题目文件夹，不得平铺到根目录。
6. `G0` 资料卡派工必须直接写明“只读原始材料，不得读取 `gold/`、样稿、成稿 `docx` 或其他答案形文档”；若原始材料不足，只允许回报 `blocked`，不得继续硬抽。
7. 自 `2026-04-06` 起，题库线默认不再派发“先做 `G0` 卡 / 先补 `JSON` 卡”的任务；默认任务改成“直接读取原始资料并落 `raw_source_analysis.md`”。如无正式反证，不得把 `G0` 卡类复杂结构重新抬成主路径。

### 4. 文档与留痕规则固定

1. 主 session 负责把角色分工、阶段纠偏、关键裁决同时回写实验中心 `00` 号文件、当前阶段正式文件和 [00_总留痕.md](../03_过程强留痕/00_总留痕.md)。
2. 子线的过程说明、派工草稿和临时清单默认不入仓长期保留；长期只保留正式入口、正式方案、正式状态卡、必要脚本与必要样板。
3. 任何并发子线的正向口径都必须带直接证据；没有同轮证据时，只能写“已尝试”或“已实现”，不得写“已验证”或“已完成”。
4. 任何关键纠偏如果只留在聊天或子线内部说明里，一律按未发生处理。

### 5. 使用边界固定

1. 这套方法论用于当前实验中心如何组织多 Agent 工作，不等于任何单次实验已经成功。
2. 如果主 session 还没有明确本轮唯一目标，就不得先开并发方案修订。
3. 如果当前问题本质上还是题目映射、变量绑定或阶段口径未冻结，就先止血收口，再开并发执行。

6. 当前口径下的 `PDF` 阅读链不是 `IID` 主实验变量，而是开跑前工程闸门；参考样稿也不是标准答案替身，而且在样稿资产、选择人、可见阶段和防泄露边界未被正式冻结前，它连条件臂都不能启动。
