3. `写作范围 * 最小内容单元 = 最小有效内容单元`
4. `写作范围 * 最小逻辑单元 = 最小有效逻辑单元`

7. `内容组合 * 逻辑组合 = 文章大纲`
8. `写作范围 * 文章大纲 = 有效大纲`

1. `动作 / 构件` 先回看能否拆成 `最小内容单元` 或 `内容组合`，不再先当天然母类。

4. `人格` 归到规则族的组合项位置，通过 `写作目的 * 人格 = 写作范围` 接入主链。
5. 范围外资产不硬塞进本轮公式体系。
