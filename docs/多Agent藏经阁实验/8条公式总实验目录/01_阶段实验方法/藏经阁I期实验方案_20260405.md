## 四、术语硬定义

1. 第三段中的 `接线`，只指把上层资产明确回填到下层组合资产、规则和最小单元；不等于新建平行框架。
2. 第三段中的 `上层资产`，优先指公式 `2~8` 已冻结的 `写作范围 / 内容组合 / 组合逻辑 / 文章大纲 / 有效大纲` 等结果位。
3. 第三段中的 `已闭合`，只指该资产已经具备可回拆、可复核、可由下层完整导出的接线关系。
4. 第三段中的 `待组合`，只指该资产已被正式承认存在，但其下层组合资产仍未接全。
5. 第三段中的 `仅候选`，只用于尚未进入共享母表、或虽被提出但证据仍不足的条目。
