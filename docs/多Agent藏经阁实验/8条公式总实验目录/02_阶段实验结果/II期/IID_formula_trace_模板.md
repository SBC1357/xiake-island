# IID formula trace 模板

状态：模板  
日期：2026-04-06  
用途：`IID / IIE / IIF` 每题必须填写的公式调用痕迹追踪表；不是结果文档，是执行模板

## 使用说明

1. 后续每做一道正式题，必须复制本模板并逐项填写
2. 本模板是 A/B 对照四步证明中 `Step 2`（公式调用痕迹）的载体
3. 如果某个公式位在当前题中不适用，必须显式标注"本题不适用 + 原因"，不得静默跳过
4. 填写完成的 trace 文件命名为 `IID_formula_trace_Q序号_题目代号.md`
