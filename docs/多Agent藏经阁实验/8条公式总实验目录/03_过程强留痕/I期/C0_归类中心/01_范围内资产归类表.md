## 当前公式主链节点

```
公式1: 写作目标 * 排列规则 = 写作目的
公式2: 写作目的 * 人格 = 写作范围
公式3: 写作范围 * 最小内容单元 = 最小有效内容单元
公式4: 写作范围 * 最小逻辑单元 = 最小有效逻辑单元
公式5: 最小有效内容单元 * 内容单元组合规则 = 内容组合
公式6: 最小有效逻辑单元 * 逻辑组合规则 = 组合逻辑
公式7: 内容组合 * 组合逻辑 = 文章大纲
公式8: 写作范围 * 文章大纲 = 有效大纲
```

可落位节点：`写作目标`、`排列规则`、`写作目的`、`人格`、`写作范围`、`最小内容单元`、`最小逻辑单元`、`内容单元组合规则`、`逻辑组合规则`、`L1表达层`、`审计证据层`、`停放观察`

## 五、本轮归类原则说明

1. `动作`不再自动充当母类。每条动作拆看它实际描述的是"意图"（归写作目的）、"手段"（归最小内容单元）还是"约束"（归规则）。
2. `构件`直接回归最小内容单元候选，不再先当框架层对象。
3. `规则`一律拆看属于排列规则、内容单元组合规则还是逻辑组合规则，不再混写成一个桶。
4. `拓扑`回到逻辑侧（最小逻辑单元或逻辑组合规则），不再单独占旧母位。
5. `人格`经由公式2接入主链，不退出也不后贴。
6. `CO-01/CO-02` 与 `ACT-05/ACT-04` 重复的，做去重处理。
7. `有限组合`和`旧容器`拆后的子部分才归位，原条目不直接归位。
8. 旧框架容器和未升格观察项统一停放，不硬塞进公式。
