## 公式位置定义

```
公式5: 最小有效内容单元 * 内容单元组合规则 = 内容组合
公式6: 最小有效逻辑单元 * 逻辑组合规则 = 组合逻辑
```

- **内容单元组合规则**：约束"最小有效内容单元之间如何拼装成内容组合"的规则。
- **逻辑组合规则**：约束"最小有效逻辑单元之间如何编排成组合逻辑"的规则。

## 一、内容单元组合规则

### 1.1 已确认规则（从第1项直接归入）

| 编号 | 规则名 | 公式作用 | 来源 |
|------|--------|---------|------|
| CR-01 | 判断必须由事实购买 | 事实单元必须在判断单元之前出现，判断才有合法性 | RUL-01 |
| CR-02 | 判断越大，细节成本越高 | 判断量级与前置细节/事实的投入量成正比 | RUL-02 |
| CR-03 | 收尾不能高于证据最大判断 | 收尾的判断天花板 = 正文已购买的最大判断 | RUL-06 |

### 1.2 从组合处理项拆出的候选规则

| 编号 | 规则名 | 公式作用 | 拆出来源 |
|------|--------|---------|---------|
| CR-04 | 失败素材必须出场 | 内容组合中必须包含失败/风险类素材，不可全正面 | RUL-05 拆件① |
| CR-05 | 引述必须拆到信息点 | 整段引述不能直接作为材料使用，必须拆到信息点/判断粒度 | CMP-05 拆件 |
| CR-06 | 场景必须拆到构件 | 整个场景不能直接作为材料使用，必须拆到其内含的事实/人物/动作 | CMP-04 拆件 |
| CR-07 | 关系必须逐条展开 | 一组关系不能直接打包使用，必须逐条独立使用 | CMP-03 拆件 |

### 1.3 内容单元组合规则的层次结构

```
第一层：准入约束
  CR-04  失败素材必须出场
  CR-05  引述必须拆到信息点
  CR-06  场景必须拆到构件
  CR-07  关系必须逐条展开

第二层：排列约束（组合后的顺序）
  CR-01  判断必须由事实购买（先事实后判断）
  CR-02  判断越大细节成本越高（梯度递进）

第三层：上限约束（组合的天花板）
  CR-03  收尾不能高于证据最大判断
```

说明：
- 第一层规则在组合开始前先生效，确保投入的单元已是原子粒度。
- 第二层规则约束原子单元之间的排列顺序和投入比例。
- 第三层规则约束组合产出的判断天花板。

### 2.3 逻辑组合规则的层次结构

```
第一层：结构选型规则（决定选哪种组合模式）
  LR-02  结构选型由内容拓扑决定

  LR-01  承重比喻必须前置
  LR-06  边界-收束联动（先边界后收束，不可逆）
  LR-07  力度调节（力度随说服路径动态调）

第三层：组合逻辑模式库（可选用的组合模式）

  │   LR-08  多步推演链
  │   LR-09  阶梯推进
  │   LR-14  分层递进
  ├── 对比/反差型
  │   LR-03  反讽式对比
  │   LR-10  双层递进
  │   LR-11  时间线对比
  │   LR-15  荒诞假设-回照
  ├── 并行/交织型
  │   LR-12  并列推进
  │   LR-13  人物-问题双线
  └── 特殊形式型
      LR-05  失败-改写联动
      LR-16  对话体推进
      LR-17  首尾意象呼应
```

## 三、两套规则的接口关系

```
内容单元组合规则（CR系列）         逻辑组合规则（LR系列）
       ↓                               ↓
   内容组合 ──────────┬────────── 组合逻辑
                      ↓
                  文章大纲（公式7）
```

- **CR系列**决定"用什么材料、按什么比例和顺序拼"。它产出的是内容组合——一组已排好的材料包。
- **LR系列**决定"用什么逻辑结构、按什么推进模式编排"。它产出的是组合逻辑——一套已选定的论证骨架。
- **公式7**把两者合成文章大纲。大纲 = 已排好的材料包 × 已选定的论证骨架。

### 3.1 CR与LR的交叉约束

1. **CR-01（判断由事实购买）** 与 **LR-09/14（阶梯/分层递进）** 交叉：每层递进都需要前置事实购买，不能空层递进。
2. **CR-04（失败素材必须出场）** 与 **LR-05/07（失败-改写联动/力度调节）** 交叉：失败素材的纳入（CR侧）和失败素材的论证处理方式（LR侧）必须协调。
3. **CR-03（收尾天花板）** 与 **LR-17（首尾意象呼应）** 交叉：意象回归收尾仍受证据天花板约束，不可用意象偷渡超额判断。

## 四、当前判断

1. 内容单元组合规则已形成"准入→排列→上限"三层结构，首轮可讨论。

3. 模式库（LR-03 ~ LR-17）较大，来自组合处理项拆件。是否全部保留为独立规则，或部分压缩为同类次型，留待后续三中心汇总讨论。
4. 两套规则当前已建立接口（公式7），但公式7的实际运作需要等第3/4项形态定义完成后才能验证。
