## 一、「文章大纲」的形态定义（公式7产出）

### 1.1 定义

**文章大纲** = 内容组合 × 组合逻辑的合成产物。它描述的是"一篇文章的骨架结构是什么样的"——材料怎么放、逻辑怎么推。

文章大纲不是一个固定模板，而是一个**结构型**：它由一种组合逻辑模式（LR系列）驱动，内含若干已排好的内容组合段落。

### 1.2 从 13 篇已分析样本提炼的大纲型族

基于 IIA期 13 篇样本的结构分析，当前可辨识的大纲型族如下：

#### 型族A：级联递推型

| 特征 | 描述 |
|------|------|
| 驱动逻辑 | LR-04（因果串接）或 LR-09（阶梯推进） |
| 骨架结构 | 起点→因果/阶梯→逐层推高→终点 |
| 段落关系 | 后段建立在前段之上，不可调序 |
| 代表样本 | A1（EGFR：旧方法失效→机制→新方案→验证→边界）、A2（小米B面：危机→突破→纵深→升级→收束）、D1（透析巨头：权威→需求→优势→空间→模式→收束） |
| 内容组合特点 | 每层都需要前置事实购买（CR-01），层级递进时细节成本递增（CR-02） |

#### 型族B：双层/多层递进型

| 特征 | 描述 |
|------|------|
| 驱动逻辑 | LR-10（双层递进）或 LR-14（分层递进） |
| 骨架结构 | 宏观线 ∥ 微观线各自递进，在某处交汇 |
| 段落关系 | 两条线内部不可调序，线间有固定交汇点 |
| 代表样本 | D2（满手死亡管线：基研薄弱(宏)→靶点受限→死亡海域→公司豪赌(微)） |
| 内容组合特点 | 宏观线用数据锤击（CO-03），微观线用风险解剖（CO-05），交汇处用判断购买 |

#### 型族C：对比/反差型

| 特征 | 描述 |
|------|------|
| 驱动逻辑 | LR-03（反讽式对比）或 LR-11（时间线对比）或 LR-15（荒诞假设-回照） |
| 骨架结构 | A面 vs B面，或正面 vs 反面，或历史 vs 当下 |
| 段落关系 | 两面构成对比张力，可先A后B或交替 |
| 代表样本 | D3（集采开倒车：当前→历史正面→现状打乱→双方对切→判断）、C1（大众点评痛揍AI：降维论证+三层防线递进） |
| 内容组合特点 | 失败素材必须出场（CR-04），力度调节随说服路径变化（LR-07） |

#### 型族D：并列推进型

| 特征 | 描述 |
|------|------|
| 驱动逻辑 | LR-12（并列推进） |
| 骨架结构 | 多条等权线平行展开，无固定先后 |
| 段落关系 | 各线相对独立，但共享一个总前提和总收束 |
| 代表样本 | D4（技术牛市：A股逻辑 ∥ H股逻辑 ∥ 出海模式） |
| 内容组合特点 | 每条线内部需独立满足事实购买（CR-01），收尾天花板（CR-03）取各线中最保守的 |

#### 型族E：人物核心轴型

| 特征 | 描述 |
|------|------|
| 驱动逻辑 | LR-13（人物-问题双线）|
| 骨架结构 | 一条人物线+一条问题线交织，围绕一个人物核心轴反复回钉 |
| 段落关系 | 每段都必须回扣核心轴，核心轴是全文锚点 |
| 代表样本 | B1（稚晖君：核心轴"急迫"→身体→情绪→失败→驱动力）、B2（朱易：被误解→证明→失控→校准→重建）、B3（沈亚楠：不满→偏执→跨域→创业） |
| 内容组合特点 | 细节素材围绕核心轴选取，非核心轴素材直接丢弃而非降级 |

#### 型族F：特殊形式型

| 特征 | 描述 |
|------|------|
| 驱动逻辑 | LR-16（对话体推进）或 LR-17（首尾意象呼应） |
| 骨架结构 | 对话/问答驱动，或首尾意象呼应结构 |
| 段落关系 | 对话体：每轮对话推进一层论证；意象型：开篇埋象→中间展开→结尾回收 |
| 代表样本 | C3（不喜欢工作：人vs猫对话→视角揭示）、C4（张曼玉金镶玉：人物锚定→回钻→矛盾→意象回归） |
| 内容组合特点 | 对话体需要短句重锤，意象型需要感官载情，都偏重表达层 |

### 1.3 大纲的形态定义

基于以上 6 个型族，文章大纲的形态可定义为：

```
文章大纲 = {
  型族标识: A|B|C|D|E|F,
  驱动逻辑: LR编号,
  段落数: n,
  段落序列: [{段落编号, 段落功能, 内容单元集, 前置依赖}],
  段落约束: {可调序: true|false, 交汇点: [位置]},
  收尾天花板: 由CR-03决定
}
```

## 二、「有效大纲」的形态定义（公式8产出）

### 2.1 定义

**有效大纲** = 写作范围 × 文章大纲。它不是一个新模板，而是"文章大纲经过写作范围筛选后的可执行版本"。

**区别**："文章大纲"是一个结构骨架；"有效大纲"是这个骨架在特定写作范围内可以被执行的实例。同一个级联递推型大纲，在"分子生物学→临床应用"的范围下和在"政策规则→产业博弈"的范围下，产出的有效大纲完全不同。

### 2.2 有效大纲的筛选机制

写作范围对文章大纲的筛选作用体现在三个层面：

| 筛选层面 | 作用 | 示例 |
|---------|------|------|
| 内容准入 | 哪些最小内容单元在此范围内可用 | A1 范围是"分子生物学→临床应用"，则金融类数据锤击不可纳入 |
| 逻辑适配 | 哪些组合逻辑模式在此范围内合理 | B1 范围是人物内核驱动，则并列推进型不适配，人物核心轴型适配 |
| 深度约束 | 范围决定判断的最大深度 | D4 是板块级投研判断，收尾不能深入到个股估值层 |

### 2.3 有效大纲的形态定义

```
有效大纲 = {
  基底大纲: 文章大纲实例,
  范围约束: 写作范围实例,
  可用内容池: [经范围筛选后的最小有效内容单元],
  可用逻辑池: [经范围筛选后的最小有效逻辑单元],
  段落实例化: [{段落编号, 实际填入的内容单元, 实际使用的逻辑推进}],
  验证: {
    CR约束全部满足: true|false,
    LR约束全部满足: true|false,
    收尾天花板检查: pass|fail
  }
}
```

### 2.4 从 13 篇样本看"有效"与"无效"的边界

| 样本 | 大纲型族 | 为什么是有效大纲 |
|------|---------|----------------|
| A1 (EGFR) | 级联递推 | 每层都有分子生物学事实购买，收尾停在框架建议而非临床指南级判断 |
| D3 (集采) | 时间线对比 | 双方对切有具体政策条文和企业行为数据支撑，不是空对空 |
| B1 (稚晖君) | 人物核心轴 | 每处回钉都有采访细节或公开信息支撑，不靠标签维持核心轴 |
| C3 (不喜欢工作) | 对话体 | 对话体形式与"猫视角揭示真相"的目的精确匹配 |

**无效大纲的反面**（推断）：
- 用级联递推型写人物稿 → 逻辑适配不通过
- 用人物核心轴型写政策分析 → 内容准入不通过（没有人物素材）
- 收尾超出正文证据天花板 → CR-03 不通过

## 三、「写作范围」的形态定义（公式2产出）

### 3.1 定义

**写作范围** = 写作目的 × 人格。它描述的是"这篇文章准备在什么深度、什么广度、什么角度上写"。

写作范围不是主题标签（如"医药"、"科技"），而是一个**空间约束**：它同时限定了内容可以探到多深、触及多广、用什么视角看。

### 3.2 从 13 篇样本提炼的写作范围维度

| 维度 | 定义 | 从样本提炼的取值范围 |
|------|------|---------------------|
| 深度轴 | 最深可以钻到哪个层面 | 分子机制(A1) → 临床结果(A1) → 商业模型(D1) → 投资判断(D4) → 人格内核(B1-B3) |
| 广度轴 | 横向覆盖几条路径 | 单线(A1/B1) → 双线(D2/B3) → 多线(D4) |
| 转换轴 | 从什么起点到什么终点 | 技术→临床(A1)、危机→蜕变(A2)、宏观→微观(D2)、规则→博弈(D3)、标签→内核(B1-B3)、具象→抽象(C2) |
| 视角轴 | 用谁的眼睛看 | 分析师(D系列)、叙事者(A2/B系列)、对话者(C3)、旁观解构者(C1/C4) |

### 3.3 写作范围的形态定义

```
写作范围 = {
  深度: {最浅层, 最深层},
  广度: {线数: n, 线间关系: 并列|交汇|主从},
  转换: {起点域, 终点域},
  视角: 分析师|叙事者|对话者|旁观解构者|...,
  人格标记: 由公式2的"人格"输入决定
}
```

### 3.4 写作范围对下游的约束传导

| 下游 | 写作范围如何约束它 |
|------|------------------|
| 公式3 → 最小有效内容单元 | 范围之外的素材不纳入（如：人物稿范围内不纳入政策推演素材） |
| 公式4 → 最小有效逻辑单元 | 范围之外的逻辑结构不适配（如：板块级范围不适配个股级联） |
| 公式8 → 有效大纲 | 文章大纲必须经过范围验证后才是有效大纲 |

### 3.5 人格在写作范围中的作用

根据公式2（`写作目的 * 人格 = 写作范围`），人格不是叠加在目的之上的装饰，而是对写作目的的**视角化具象**：

- 同一个写作目的（如"解释为什么某个旧方法失效了"），经过不同人格处理后，写作范围完全不同：
  - 分析师人格 → 范围是"数据→机制→替代方案"（A1路线）
  - 叙事者人格 → 范围是"人的经历→踩坑→蜕变"（A2路线）
  - 旁观解构者人格 → 范围是"荒诞→拆解→真相"（C1路线）

## 四、三者的关系总览

```
写作目的 × 人格 ──→ 写作范围
                        │
                ┌───────┴───────┐
                ↓               ↓
          最小有效内容单元   最小有效逻辑单元
                │               │
         (内容组合规则)    (逻辑组合规则)
                │               │
                ↓               ↓
           内容组合        组合逻辑
                └───────┬───────┘
                        ↓
                   文章大纲（6种型族）
                        │
              写作范围 × 文章大纲
                        ↓
                   有效大纲（可执行实例）
```

## 五、当前判断

1. **文章大纲已形成6种型族**：级联递推、双层递进、对比反差、并列推进、人物核心轴、特殊形式。13篇样本全部可归入这6种之一。
2. **有效大纲的筛选机制已定义三层**：内容准入、逻辑适配、深度约束。
3. **写作范围已形成4维描述**：深度轴、广度轴、转换轴、视角轴，加上人格标记。
4. **人格的位置已明确**：人格不是装饰，它对写作目的做视角化具象，直接决定写作范围的视角轴。
5. **尚未验证**：以上形态定义来自 13 篇已分析样本，未在 32 篇全量样本上检验。第 5 项推演可能会补充新的型族或修正现有定义。
