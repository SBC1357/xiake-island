## 一、写作目标候选类型

写作目标是公式链最上游输入。从 32 篇样本中归纳出以下 8 种候选类型：

### OBJ-01 还原决策逻辑

**定义**：向读者还原一个关键决策（监管审批、治疗路径选择、商业判断等）背后的底层逻辑，让读者从"知道发生了什么"推进到"理解为什么必须这样"。

**典型特征**：存在一个明确的决策事件作为锚点；文章的核心价值不在报道决策本身，而在拆解决策的逻辑链。

### OBJ-02 赋予判断标尺

**定义**：为读者构建一套可操作的评价框架或比较工具，让读者从"不知道该怎么选"推进到"拥有判断框架"。

**典型特征**：存在一个需要比较或评价的对象集合；文章的核心价值是交付一把尺子，而非代替读者下结论。

### OBJ-03 重建认知坐标

**定义**：推翻或升级读者对某个领域/疾病/技术的原有认知坐标系，让读者从"旧地图"迁移到"新地图"。

**典型特征**：存在一个明确的"旧认知"作为出发点；文章的核心价值是完成坐标系位移，而非信息增量。

### OBJ-04 解码战略路径

**定义**：拆解一个商业实体的战略选择与竞争逻辑，让读者从"看到动作"推进到"看透战略"。

**典型特征**：存在一个以公司/管线/品牌为主体的分析单元；核心价值是揭示战略的内在一致性。

### OBJ-05 还原人物内核

**定义**：穿透人物的表层行动和公开叙事，还原其核心决策驱动力，让读者从"认识一个人"推进到"理解一个人为什么做这个选择"。

**典型特征**：存在一个明确的人物作为叙事中心；核心价值是从行为反推驱动力。

### OBJ-06 机制→价值翻译

**定义**：把一个技术/机制层面的创新拆解到"为什么对临床重要"这一层，让读者从"知道它是什么"推进到"理解它为什么重要"。

**典型特征**：存在一个需要解释的技术/机制创新；核心价值是完成从机制到临床价值的翻译。

### OBJ-07 证据→判断转译

**定义**：将临床试验数据翻译成可供临床决策参考的结论，让读者从"看到数据"推进到"能做判断"。

**典型特征**：存在一个或多个临床研究结果作为核心素材；核心价值是把数据转化为有方向的判断。

### OBJ-08 非常规照亮

**定义**：用一个意外的、跨界的、非标准的切入角度重新照亮一个读者自认为已经熟悉的话题。

**典型特征**：起章切入点与行业常规叙事明显不同（借喻、类比、跨界概念）；核心价值是通过认知裂缝制造重新审视的契机。

## 二、写作目的候选类型

写作目的 = 写作目标 × 排列规则。以下候选类型描述的是 32 篇样本中可归纳的"目标 × 排列"组合产出：

### PUR-01 裁决权转移

**定义**：通过系统比较 / 框架构建 / 尺子交付，使读者获得某个领域的自主判断权。

**上游写作目标**：主要由 OBJ-02（赋予判断标尺）驱动，也可由 OBJ-01（还原决策逻辑）与评价框架排列结合产出。

**排列特征**：先建标准，再放入比较对象，最后让读者自己走一遍判断流程。读者读完后获得的不是一个结论，而是一个可复用的判断方法。

### PUR-02 坐标系重置

**定义**：通过底层逻辑揭示和范式对比，完成读者认知坐标系的不可逆位移。

**上游写作目标**：主要由 OBJ-03（重建认知坐标）驱动。

**排列特征**：先确认旧坐标系的合理性边界，再揭示其失效条件，最后展示新坐标系的自洽性。位移完成后读者无法回到旧认知。

### PUR-03 决策必然性还原

**定义**：通过层层剥开，让读者理解一个已经发生/即将发生的关键决策为什么是必然的。

**上游写作目标**：可由 OBJ-01（还原决策逻辑）或 OBJ-04（解码战略路径）驱动——前者侧重制度/科学决策，后者侧重商业/战略决策。

**排列特征**：以决策事件为锚点，向上追溯约束条件、向下推演结果必然性。读者读完后的反应是"原来这个决定不得不这样做"。

### PUR-04 证据→判断管道

**定义**：通过结构化的数据解读流水线，将临床证据翻译成可操作的临床判断。

**上游写作目标**：主要由 OBJ-07（证据→判断转译）驱动，也可由 OBJ-06（机制→价值翻译）与证据型排列结合产出。

**排列特征**：先框定研究背景和未满足需求，再拆解数据的正确读法，最后导出具有方向性的临床判断。核心排列动作是"数据不等于显而易见的结论，我来告诉你正确的读法"。

### PUR-05 人物驱动力解码

**定义**：通过行为→动机的递推，让读者触碰到人物选择背后的核心驱动力。

**上游写作目标**：由 OBJ-05（还原人物内核）驱动。

**排列特征**：以公开行为/业绩/选择为表层素材，通过筛选关键决策节点、追溯约束条件、对比放弃的替代方案，反推出驱动力本体。读者读完后的反应是"我现在理解他为什么做这个选择了"。

### PUR-06 机制价值化

**定义**：通过机制→临床→实践的多级翻译，让读者理解一项技术创新的真实价值层级。

**上游写作目标**：由 OBJ-06（机制→价值翻译）驱动。

**排列特征**：从分子/技术机制起步，逐级翻译到临床获益、再翻译到实践层面的意义变化。每一级翻译都是一次"所以呢？"的追问和回答。

### PUR-07 反常识引爆

**定义**：通过非标准切入点制造认知裂缝，然后用专业内容填充裂缝，完成对已知话题的重新审视。

**上游写作目标**：由 OBJ-08（非常规照亮）驱动。

**排列特征**：开篇刻意制造与行业惯常叙事的偏差（借喻、跨界、数据反直觉），利用偏差产生的注意力窗口导入专业内容。读者读完后的反应是"这个角度我从来没想过"。

## 三、候选排列关系

### 3.1 写作目标→写作目的的映射规律

| 规律编号 | 描述 |
|---------|------|
| MAP-01 | **一对一主通道**：多数情况下一种写作目标有一条"阻力最小"的排列路径，对应一种写作目的。OBJ-03→PUR-02、OBJ-05→PUR-05、OBJ-08→PUR-07 属于强绑定。 |
| MAP-02 | **同目标多目的**：同一写作目标在不同排列规则下可产出不同写作目的。最典型的是 OBJ-06（机制→价值翻译），既可走 PUR-06（机制价值化——逐级翻译排列），也可走 PUR-04（证据→判断管道——数据流水线排列）。 |
| MAP-03 | **同目的多源头**：不同写作目标可以通过不同排列规则汇聚到同一种写作目的。PUR-03（决策必然性还原）既可由 OBJ-01（制度/科学决策逻辑）驱动，也可由 OBJ-04（商业战略路径）驱动。 |

### 3.2 排列规则的角色观察

从 32 篇样本中可归纳出排列规则在"目标→目的"转换中的以下作用模式：

| 编号 | 排列作用模式 | 说明 |
|------|-------------|------|
| ARR-01 | 逐级翻译 | 从一个领域的语言翻译到另一个领域，每一级的"所以呢？"推动写作目标向写作目的转化。多见于 OBJ-06→PUR-06。 |
| ARR-02 | 框架先行 | 先建标准再放入比较对象，排列的核心动作是"交付尺子"而非"给出结论"。多见于 OBJ-02→PUR-01。 |
| ARR-03 | 层层剥开 | 以事件/决策为锚点，向上追溯约束链。排列的核心动作是"还原约束条件的累积"。多见于 OBJ-01/OBJ-04→PUR-03。 |
| ARR-04 | 旧→新对位 | 先确认旧坐标系边界，再揭示失效条件，最后展示新坐标系。排列的核心动作是"制造不可逆的认知位移"。多见于 OBJ-03→PUR-02。 |
| ARR-05 | 行为反推 | 以公开行为为表层，筛选关键决策节点，追溯至驱动力本体。排列的核心动作是"从外向内递推"。多见于 OBJ-05→PUR-05。 |
| ARR-06 | 数据管道 | 先框定未满足需求，再拆解数据正确读法，最后导出方向性判断。排列的核心动作是"建立从数据到判断的单行道"。多见于 OBJ-07→PUR-04。 |
| ARR-07 | 偏差引入 | 开篇制造与行业惯常叙事的偏差，利用偏差窗口导入专业内容。排列的核心动作是"先破后立"。多见于 OBJ-08→PUR-07。 |

### 3.3 目标→排列→目的全景映射

```
OBJ-01 还原决策逻辑 ──┬── ARR-03 层层剥开 ──→ PUR-03 决策必然性还原
                       └── ARR-02 框架先行 ──→ PUR-01 裁决权转移（当决策涉及多方博弈时）

OBJ-02 赋予判断标尺 ──── ARR-02 框架先行 ──→ PUR-01 裁决权转移

OBJ-03 重建认知坐标 ──── ARR-04 旧→新对位 ──→ PUR-02 坐标系重置

OBJ-04 解码战略路径 ──── ARR-03 层层剥开 ──→ PUR-03 决策必然性还原

OBJ-05 还原人物内核 ──── ARR-05 行为反推 ──→ PUR-05 人物驱动力解码

OBJ-06 机制→价值翻译 ─┬─ ARR-01 逐级翻译 ──→ PUR-06 机制价值化
                       └─ ARR-06 数据管道 ──→ PUR-04 证据→判断管道（当机制文章以数据为核心论据时）

OBJ-07 证据→判断转译 ──── ARR-06 数据管道 ──→ PUR-04 证据→判断管道

OBJ-08 非常规照亮 ──── ARR-07 偏差引入 ──→ PUR-07 反常识引爆
```

## 四、与前项产出的交叉观察

### 4.1 填补写作目标节点空缺

第1项归类结论曾指出"写作目标节点空缺"。本项推演出 8 种写作目标候选类型，可作为该节点的首轮候选资产。其中 OBJ-01 ~ OBJ-08 均为从 32 篇样本中归纳的抽象类型，不与任何单篇绑定。

### 4.2 与排列规则候选的对照

第1项归类中确认的 2 条排列规则候选（ACT-06 细节买权、ARCH-05 可迁移的是结构作用）与本项 ARR-01 ~ ARR-07 的关系：

- **ACT-06（细节买权）** 更像一条贯穿所有排列模式的通用原则——无论哪种排列，都需要用细节积累说服力来"购买"做大判断的权利。它不是一种独立排列模式，而是所有排列模式的质量约束。
- **ARCH-05（可迁移的是结构作用）** 与本项的发现一致：32 篇样本的可迁移价值不在具体素材，而在排列结构本身。ARR-01 ~ ARR-07 正是对"结构作用"的首轮具象化。

### 4.3 与写作范围形态定义的对照

第3+4项结论指出"写作范围的转换轴是关键特征"。本项发现写作目标天然包含一个转换方向（每个 OBJ 都描述了"从 A 到 B"的认知位移目标），这与写作范围的转换轴形成映射——写作目标定义了转换的方向，写作范围定义了转换的允许域。
