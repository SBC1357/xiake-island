## 当前归位原则

1. `动作 / 构件 / 规则 / 拓扑` 不再作为母口径，只能作为来源标签。
2. 结构上同一单元、来源上双入口的条目，主链只占一个槽位，但必须保留双入口留痕。
3. `有限组合` 与 `旧容器` 一律不按原条目归位，只按拆件后的子部分回填。
4. `人格` 不再继续停放观察，直接通过公式 `2` 接回主链。
5. `旧框架容器` 与 `未升格观察项` 只保留证据位，不硬塞进本轮公式体系。

## 当前公式主链

```text
公式1：写作目标 * 排列规则 = 写作目的
公式2：写作目的 * 人格 = 写作范围
公式3：写作范围 * 最小内容单元 = 最小有效内容单元
公式4：写作范围 * 最小逻辑单元 = 最小有效逻辑单元
公式5：最小有效内容单元 * 内容单元组合规则 = 内容组合
公式6：最小有效逻辑单元 * 逻辑组合规则 = 组合逻辑
公式7：内容组合 * 组合逻辑 = 文章大纲
公式8：写作范围 * 文章大纲 = 有效大纲
```
