## 二、文章大纲形态

这里的 `文章大纲`，按当前主链理解，是 `内容组合 * 组合逻辑` 的产物；它先回答“这篇文章怎么走”，还不回答“在什么范围里才有效”。

| 形态编号 | 当前形态 | 结构骨架 | 主要证据来源 |
|----------|----------|----------|--------------|
| OUT-01 | 尺子前置-分层递进型 | 先立评价尺子 / 判断坐标 -> 再逐层拆事实、风险、比较对象 -> 最后回收到结论 | `IIE` 样本 `1/2`；[01_批次A_盲复核结果.md](../../../../02_阶段结果/IIE期/C1_结构复核中心/01_批次A_盲复核结果.md) |
| OUT-02 | 重定义-收束主张型 | 先证明旧分类 / 旧赛道不够 -> 再用分类、结构、临床或平台证据推进 -> 最后收回“唯一 / 标杆 / 差异化”主张 | `IIE` 样本 `3/4`；[01_批次A_领域复核卡_盲复核版.md](../../../../02_阶段结果/IIE期/C2_领域复核中心/01_批次A_领域复核卡_盲复核版.md) |
| OUT-03 | 人物谜题-真相回钻型 | 标题先钉“这个人为何如此 / 为何做成” -> 中段以技术、行业、经历做证据插层 -> 结尾回到人物行动逻辑 | `IIE` 样本 `5`；[01_批次A三路分歧对照表.md](../../../../02_阶段结果/IIE期/01_批次A三路分歧对照表.md) |
| OUT-04 | 需求-解法对照型 | 先列需求 / 标准 -> 再逐项验证解法是否满足 -> 最后回到工程或临床意义 | `IIE` 样本 `6`；[01_批次A_领域复核卡_盲复核版.md](../../../../02_阶段结果/IIE期/C2_领域复核中心/01_批次A_领域复核卡_盲复核版.md) |
| OUT-05 | 结果前置-来源还原-边界界定型 | 先承认阳性结果或显著结论 -> 再逆向拆来源、人群、场景 -> 最后落到边界和分流建议 | `IIE` 样本 `7`；[01_批次A三路分歧对照表.md](../../../../02_阶段结果/IIE期/01_批次A三路分歧对照表.md) |

### 当前不单列为通用文章大纲的情况

`SUPP-PROJ-002` 虽然已经被深分析，但当前更像“有效大纲和写作范围的边界样本”，而不是已经稳定成型的通用 `文章大纲`。原因是它在现框架里的主路线仍未被完全压平，强行升格只会把“患者沟通型安全比较”提前写成新口径。

## 五、写作范围形态

这里的 `写作范围`，按当前主链理解，是 `写作目的 * 人格` 的结果。它不是题材目录，而是“这篇文章允许采用什么判断姿态、服务什么任务”的工作边界。

| 范围编号 | 当前范围 | 目的侧 | 人格侧 | 主要证据来源 |
|----------|----------|--------|--------|--------------|
| RANGE-01 | 评价-解释 / 评价-工具 / 评价-建构范围 | 建立判断尺子、解释分歧、组织比较、构建评价框架 | 分析型、评判型、建构型人格 | `IIE` 样本 `1/2/4`；[01_批次A_盲复核结果.md](../../../../02_阶段结果/IIE期/C1_结构复核中心/01_批次A_盲复核结果.md) |
| RANGE-02 | 主张-论证范围 | 把“唯一 / 最优 / 标杆 / 差异化”锚定出来 | 强主张人格、论证人格 | `IIE` 样本 `3`；[01_批次A_领域复核卡_盲复核版.md](../../../../02_阶段结果/IIE期/C2_领域复核中心/01_批次A_领域复核卡_盲复核版.md) |
| RANGE-03 | 人物真相探索范围 | 回答“这个人到底凭什么 / 为何做成” | 人物真相钻头、现场型人物人格 | `IIE` 样本 `5`；[01_批次A三路分歧对照表.md](../../../../02_阶段结果/IIE期/01_批次A三路分歧对照表.md) |
| RANGE-04 | 工程解释降门槛范围 | 把复杂机制、结构、设计讲到可理解 | 工程解释人格，而不是比喻表演人格 | `IIE` 样本 `6`；[02_批次A规则边界待压缩清单.md](../../../../02_阶段结果/IIE期/02_批次A规则边界待压缩清单.md) |
| RANGE-05 | 决策辅助范围 | 一支是边界澄清，一支是患者 / 读者选择辅助 | 审慎评判型、患者沟通型、分析-约束型人格 | `IIE` 样本 `7/8`；[07_辰公子医学样本抽样框架与首轮候选集.md](../../../../03_阶段过渡/IID-后续推进中间环节/07_辰公子医学样本抽样框架与首轮候选集.md) |

### 当前范围判断

1. `范围` 不是“疾病 / 公司 / 人物 / 机制”的题材名，而是“这篇文章要完成什么任务、允许采取什么判断姿态”。
2. `RANGE-01` 与 `RANGE-02` 最容易混淆：前者以建立尺子和比较为主，后者以抢占定义权和收束主张为主。
3. `RANGE-05` 当前内部已有两支：`结果边界澄清` 与 `患者选择沟通`；两者同属决策辅助，但不能直接混写成一个单一路线。

3. 如果第 `5` 项补出的 `写作目标 / 写作目的` 改写了上游任务分布，这张对位表必须回写。
