## 二、写作目标候选类型

| 目标候选 | 当前理解 |
|----------|----------|
| 结果解义 | 把“阳性 / 阴性 / 获批 / 推迟 / 高票通过”翻成真正含义 |
| 风险边界 | 讲清谁获益、谁不适合、代价在哪里 |
| 机制搭桥 | 把机制、统计、研究设计、指标关系降维成可理解逻辑链 |
| 格局重绘 | 把单点结果放回疾病、赛道、线位、同类竞争的全景里 |
| 选择导航 | 给人群分层、方案优选、治疗时机、前后线衔接提供路径 |
| 标杆塑形 | 塑造产品、公司、平台、管线的“唯一性 / 领先性 / 里程碑性” |
| 人物载道 | 借人物经历、专访和现场感外显研发方法论、行业逻辑、使命感 |
| 患者沟通 | 把副作用、长期管理、生活质量、经济负担翻译成患者语言 |
| 借势纠偏 | 先用热点、比喻、公共争议开门，再把认知拉回专业判断 |

## 三、写作目的候选类型

| 目的候选 | 当前理解 |
|----------|----------|
| 形成判断 | 让读者不只知道发生了什么，还知道该怎么看 |
| 更新预期 | 把过度乐观或过度悲观拉回现实边界 |
| 推动分层使用 | 把“给谁用、何时用、怎么接”说清楚 |
| 争夺定义权 | 让某疗法、产品、公司占住“新标准 / 新阶段 / 唯一解”的位置 |
| 建立可信度 | 通过承认局限与风险，换取更强信任 |
| 增强本土信心 | 为中国监管、国产创新、国内实践争取解释权和正当性 |
| 降低焦虑 | 缓解患者、家属、从业者面对复杂选择时的无措感 |
| 放大传播记忆 | 让复杂医学议题更容易被记住、转述、复用 |
| 输出方法论 | 把成功路径、创新逻辑、研发观念抽成可迁移范式 |

## 四、目标 × 排列规则 => 目的

| 目标 | 当前排列关系 | 当前生成的目的 |
|------|--------------|----------------|
| 结果解义 | 先抛反常或冲突结果，再拆机制或统计，最后落适用边界 | 形成判断 |
| 风险边界 | 先承认问题与代价，再给风险-获益证据，末段按人群分流 | 更新预期 + 推动分层使用 |
| 格局重绘 | 先回顾旧格局或历史失败，再插入新证据与同类对照，结尾抬出新定义 | 争夺定义权 |
| 标杆塑形 / 人物载道 | 先立人或立公司，再展开方法论与路线，最后回到患者或行业价值 | 建立可信度 + 输出方法论 |
| 患者沟通 | 先代入痛点，再把副作用与长期获益翻成日常语言，结尾给出选择建议 | 降低焦虑 + 推动规范使用 |
| 借势纠偏 | 先用热点 / 比喻 / 争议开门，中段完成知识降维，末段挂回医学主问题或本土立场 | 放大传播记忆 + 增强本土信心 |
