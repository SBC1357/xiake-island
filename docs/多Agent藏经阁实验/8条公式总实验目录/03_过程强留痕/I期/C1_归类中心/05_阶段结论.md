## 第2项结论：组合规则推演

### 2.1 结论概述

当前已经可以先冻结一版双侧组合规则：

- 内容单元组合规则 `5` 条
- 逻辑组合规则 `5` 条

这 `10` 条不是最终定稿，但已经足够支撑第 `3/4` 项继续往下定义形态。

### 2.2 关键发现

1. 内容侧最稳的不是“多写点数据”，而是形成一条可检验的购买链：`事实 / 数据 / 边界 -> 判断 -> 收尾`。
2. 逻辑侧最稳的不是具体稿型名，而是三步顺序：`先立尺子 -> 再选拓扑 -> 再做对比 / 分流 / 推演`。
3. `评价尺子前置`、`边界先入链`、`收尾不能抬高`，已经从旧规则描述变成了当前可执行的组合约束。
4. 多步来源仍然只能拆件用：`历史回望链`、`政策推演`、`人物真相钻头`、`分层递进论证` 这些都还不能整包入表。

### 2.3 对后项的影响

- 第 `3` 项的大纲定义，必须把“内容组合”与“组合逻辑”分开写，再看它们如何扣合。
- 第 `4` 项的写作范围定义，必须把 `人格` 与“允许调用哪些组合规则”一起考虑。
- 第 `5` 项如果补出新的上游 `写作目标`，后续要回看这里的 `排列规则` 是否需要补强。

## 第3项结论：文章大纲 / 有效大纲形态

### 3.1 结论概述

当前已经可以先冻结两组形态：

- `文章大纲` 形态 `5` 种
- `有效大纲` 形态 `5` 种

这组形态只基于已经深分析过的文章，不依赖 `32` 篇全量样本池。

### 3.2 关键发现

1. `文章大纲` 已经不是单一“分析型”可以兜住，当前至少分出：尺子前置型、重定义主张型、人物谜题型、需求-解法型、结果边界型。

3. `SUPP-PROJ-002` 的价值现在主要在暴露边界，而不是稳定提供一条新的通用文章大纲。

### 3.3 对后项的影响

## 第4项结论：写作范围形态

### 4.1 结论概述

当前已经可以先冻结 `5` 种 `写作范围` 形态，并把它们和第 `3` 项的大纲形态做首轮对位。

### 4.2 关键发现

1. `写作范围` 不是题材分类，而是“这篇文章要完成什么任务 + 允许用什么人格姿态”的边界。
2. 当前最稳的 `5` 类范围分别是：评价-解释 / 工具 / 建构，主张-论证，人物真相探索，工程解释降门槛，决策辅助。

### 4.3 对后项的影响

- 第 `5` 项读完 `32` 篇后，如果上游 `写作目标 / 写作目的` 出现新的稳定分布，必须回写本轮的范围定义。

## 第7项结论：逐项接线冻结

### 7.1 结论概述

第三段已经按 `文章大纲 -> 有效大纲 -> 写作范围 -> 内容组合 / 组合逻辑 -> 最小单元` 的固定顺序，把 `C1` 当前的冻结资产逐项反推并接到下层。

- `文章大纲`：`4` 个 `已闭合`，`1` 个 `待组合`
- `有效大纲`：`3` 个 `已闭合`，`2` 个 `待组合`
- `写作范围`：`3` 个 `已闭合`，`2` 个 `待组合`
- `内容组合`：`4` 个 `已闭合`，`1` 个 `待组合`
- `组合逻辑`：`4` 个 `已闭合`，`1` 个 `待组合`

### 7.2 关键发现

1. `人物真相` 线当前仍依赖 `GS-01` 拆出来的 `人物线 / 问题线主从`，这条最小逻辑单元还没有被冻结成共享正式件，所以 `OUT-03 / E-OUT-03 / RANGE-03` 必须从“已收出形态”改判成 `待组合`。

3. 当前未闭合面不是因为规则条数不够，而是因为最小逻辑单元还没压平；所以本轮不新增正式规则，也不恢复旧 `II期` 母框架。

### 7.3 对后项的影响

- `Codex` 总协调如果要做共享接线母表，应直接以 [07_逐项接线冻结.md](07_逐项接线冻结.md) 的闭合/待组合判定为 `C1` 证据，不应回退到首轮只看形态数量。
