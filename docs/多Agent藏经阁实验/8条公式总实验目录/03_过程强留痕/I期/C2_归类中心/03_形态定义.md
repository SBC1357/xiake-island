## 一、相关公式回顾

```
公式3: 写作范围 * 最小内容单元 = 最小有效内容单元
公式4: 写作范围 * 最小逻辑单元 = 最小有效逻辑单元
公式5: 最小有效内容单元 * 内容单元组合规则 = 内容组合

公式7: 内容组合 * 组合逻辑 = 文章大纲
公式8: 写作范围 * 文章大纲 = 有效大纲
```

## 二、「文章大纲」形态定义

### 2.1 文章大纲的公式定位

```
公式7: 内容组合 * 组合逻辑 = 文章大纲
```

**文章大纲 = 内容组合 × 组合逻辑**

- **内容组合**：由最小有效内容单元按内容单元组合规则拼装而成的内容块
- **组合逻辑**：由最小有效逻辑单元按逻辑组合规则串联而成的逻辑链
- **文章大纲**：内容组合与组合逻辑的结合，形成文章的结构骨架

### 2.2 文章大纲的形态要素

根据公式7，文章大纲应包含以下形态要素：

| 要素 | 来源 | 说明 |
|------|------|------|
| 内容块序列 | 内容组合 | 文章包含哪些内容块 |
| 逻辑链结构 | 组合逻辑 | 内容块之间如何串联 |
| 块内结构 | 内容组合内部 | 每个内容块内部如何组织 |
| 链间关系 | 组合逻辑内部 | 逻辑链之间的嵌套、并列关系 |

### 2.3 文章大纲的形态分类

基于已分析的文章，文章大纲可分为以下几种形态：

#### 2.3.1 级联型大纲

```
【形态特征】
- 内容块按因果或递进关系串联
- 逻辑链为单线级联结构
- 典型拓扑：阶梯推进

【适用场景】
- 循证解读类文章
- 机制解释类文章
- 监管事件解读类文章

【示例结构】
背景问题 → 证据呈现 → 结果解释 → 格局判断
```

#### 2.3.2 平行型大纲

```
【形态特征】
- 内容块按并列关系排列
- 逻辑链为多线并行结构
- 典型拓扑：并列推进、多药比较

【适用场景】
- 多药比较类文章
- 治疗格局综述类文章
- 赛道全景类文章

【示例结构】
评价框架建立 → 药物A分析 → 药物B分析 → 药物C分析 → 综合判断
```

#### 2.3.3 双层递进型大纲

```
【形态特征】
- 内容块按双层结构组织
- 逻辑链为"外层 + 内层"嵌套结构
- 典型拓扑：双层递进

【适用场景】
- 公司画像类文章
- 管线矩阵类文章
- 人物-问题混合类文章

【示例结构】
外层：产业判断 → 公司定位 → 管线展开 → 标杆收束
内层：每段内部再做细节展开
```

#### 2.3.4 时间线型大纲

```
【形态特征】
- 内容块按时间顺序排列
- 逻辑链为时间轴结构
- 典型拓扑：时间线对比

【适用场景】
- 事件回望类文章
- 历史回顾类文章
- 政策推演类文章

【示例结构】
事件起手 → 历史节点1 → 历史节点2 → 当前状态 → 前瞻判断
```

### 2.4 文章大纲的C2定义

**文章大纲 = 文章的结构骨架，由内容块序列和逻辑链结构共同定义。**

形态要素：
1. **内容块序列**：文章包含哪些内容块（what）
2. **逻辑链结构**：内容块如何串联（how）
3. **块内结构**：每个内容块的内部组织（detail）
4. **链间关系**：逻辑链之间的嵌套、并列关系（relation）

## 三、「有效大纲」形态定义

### 3.1 有效大纲的公式定位

```
公式8: 写作范围 * 文章大纲 = 有效大纲
```

**有效大纲 = 写作范围 × 文章大纲**

- **写作范围**：由写作目的和人格共同限定（公式2输出）
- **文章大纲**：由内容组合和组合逻辑共同定义（公式7输出）
- **有效大纲**：在写作范围内有效的文章大纲，即"符合写作范围约束的文章结构"

### 3.2 有效大纲与文章大纲的区别

| 维度 | 文章大纲 | 有效大纲 |
|------|----------|----------|
| 定义 | 内容组合 × 组合逻辑 | 写作范围 × 文章大纲 |
| 关注点 | 文章的结构骨架 | 符合范围约束的结构骨架 |
| 判断标准 | 是否有清晰的内容块和逻辑链 | 是否在范围内、是否符合人格、是否服务于写作目的 |
| 功能 | 描述文章"长什么样" | 判断文章"是否有效" |

### 3.3 有效大纲的判断标准

根据公式8，有效大纲需满足以下条件：

1. **范围符合**：文章大纲的内容块序列在写作范围内
2. **人格符合**：文章大纲的语气、视角、风格符合人格设定
3. **目的服务**：文章大纲服务于写作目的（公式1输出）

### 3.4 有效大纲的C2定义

**有效大纲 = 在写作范围内有效的文章结构骨架。**

判断标准：
1. 内容块序列在写作范围内
2. 逻辑链结构符合人格设定
3. 整体结构服务于写作目的

## 四、「写作范围」形态定义

### 4.1 写作范围的公式定位

```
公式2: 写作目的 * 人格 = 写作范围
```

**写作范围 = 写作目的 × 人格**

- **写作目的**：由写作目标和排列规则共同定义（公式1输出）
- **人格**：写作的主体风格设定
- **写作范围**：写作目的和人格共同限定的写作边界

### 4.2 写作范围的形态要素

根据公式2，写作范围应包含以下形态要素：

| 要素 | 来源 | 说明 |
|------|------|------|
| 主题边界 | 写作目的 | 写什么、不写什么 |
| 素材边界 | 写作目的 | 用什么素材、不用什么素材 |
| 判断边界 | 写作目的 | 下什么判断、不下什么判断 |
| 风格边界 | 人格 | 什么风格、什么语气、什么视角 |

### 4.3 写作范围的形态分类

基于已分析的文章，写作范围可分为以下几种形态：

#### 4.3.1 单药评价范围

```
【写作目的】评价单一药物的价值和定位
【人格】评价尺子型分析者
【范围特征】
- 主题：聚焦单一药物的疗效、安全性、机制
- 素材：临床数据、机制研究、专家观点
- 判断：疗效判断、安全性判断、格局判断
- 风格：数据驱动、客观分析
```

#### 4.3.2 多药比较范围

```
【写作目的】比较多个药物的优劣
【人格】评价尺子型分析者
【范围特征】
- 主题：多药比较、疗效对比、格局判断
- 素材：头对头数据、间接比较、专家观点
- 判断：疗效对比判断、适应症划分判断
- 风格：评价框架先行、公平比较
```

#### 4.3.3 赛道综述范围

```
【写作目的】全景展示一个治疗领域的发展
【人格】赛道全景解读者
【范围特征】
- 主题：疾病背景、历史失败链、当前格局、前瞻催化剂
- 素材：历史数据、管线信息、专家观点
- 判断：赛道成熟度判断、竞争格局判断
- 风格：全景视角、历史纵深
```

#### 4.3.4 监管解读范围

```
【写作目的】解读监管事件背后的逻辑
【人格】制度差异解读者
【范围特征】
- 主题：监管事件、制度背景、审批逻辑
- 素材：审批文件、专家观点、历史先例
- 判断：监管逻辑判断、本土适应性判断
- 风格：制度视角、比较分析
```

#### 4.3.5 人物专访范围

```
【写作目的】通过人物展示方法论或故事
【人格】记者在场型问题采访者
【范围特征】
- 主题：人物经历、方法论、行业观点
- 素材：采访内容、背景资料
- 判断：人物价值判断、方法论评价
- 风格：在场感、问题导向
```

#### 4.3.6 公司画像范围

```
【写作目的】评价公司价值和定位
【人格】产业评价型分析者
【范围特征】
- 主题：公司战略、管线布局、产业定位
- 素材：公司资料、管线数据、行业数据
- 判断：公司价值判断、管线价值判断
- 风格：产业视角、标杆对标
```

### 4.4 写作范围的C2定义

**写作范围 = 写作目的和人格共同限定的写作边界。**

形态要素：
1. **主题边界**：写什么、不写什么
2. **素材边界**：用什么素材、不用什么素材
3. **判断边界**：下什么判断、不下什么判断
4. **风格边界**：什么风格、什么语气、什么视角

## 五、三者的关系图

```
【公式链关系】

写作目标 * 排列规则 = 写作目的
写作目的 * 人格 = 写作范围
写作范围 * 最小内容单元 = 最小有效内容单元
写作范围 * 最小逻辑单元 = 最小有效逻辑单元
最小有效内容单元 * 内容单元组合规则 = 内容组合
最小有效逻辑单元 * 逻辑组合规则 = 组合逻辑
内容组合 * 组合逻辑 = 文章大纲
写作范围 * 文章大纲 = 有效大纲

【三者的层次关系】

写作范围（边界层）
    ↓ 约束
文章大纲（结构层）
    ↓ 验证
有效大纲（有效性层）
```

## 六、C2独立判断与发现

### 6.1 文章大纲的形态分类与拓扑分类的关系

- 文章大纲的形态分类（级联型、平行型、双层递进型、时间线型）与CO-07的拓扑分类（阶梯推进、双层递进、时间线对比、并列推进）高度对应
- 这说明拓扑分类可以看作是文章大纲形态的分类依据
- **建议：将CO-07的拓扑分类与文章大纲形态分类统一整合**

### 6.2 有效大纲的关键在于"范围符合"

- 有效大纲的判断核心在于"是否在写作范围内"
- 这要求写作范围必须清晰定义（主题边界、素材边界、判断边界、风格边界）
- **第5项需要在32篇推演中验证：写作范围是否足够清晰？**

### 6.3 写作范围的形态与人格高度相关

- 不同人格对应不同的写作范围形态
- 例如："评价尺子型分析者"对应"单药评价范围"或"多药比较范围"
- "赛道全景解读者"对应"赛道综述范围"
- **人格与写作范围的映射关系需要在第5项中验证**

### 6.4 形态定义的局限

- 当前形态定义基于"已经分析过的文章"，而非32篇全量样本
- 形态分类可能不完整，需要在第5项中补充
- 形态要素可能不够精确，需要在第5项中修正

## 七、形态定义总结

### 7.1 文章大纲定义

**文章大纲 = 文章的结构骨架，由内容块序列和逻辑链结构共同定义。**

形态分类：
- 级联型大纲：内容块按因果或递进关系串联
- 平行型大纲：内容块按并列关系排列
- 双层递进型大纲：内容块按双层结构组织
- 时间线型大纲：内容块按时间顺序排列

### 7.2 有效大纲定义

**有效大纲 = 在写作范围内有效的文章结构骨架。**

判断标准：
- 内容块序列在写作范围内
- 逻辑链结构符合人格设定
- 整体结构服务于写作目的

### 7.3 写作范围定义

**写作范围 = 写作目的和人格共同限定的写作边界。**

形态要素：
- 主题边界：写什么、不写什么
- 素材边界：用什么素材、不用什么素材
- 判断边界：下什么判断、不下什么判断
- 风格边界：什么风格、什么语气、什么视角

形态分类：
- 单药评价范围、多药比较范围、赛道综述范围
- 监管解读范围、人物专访范围、公司画像范围
