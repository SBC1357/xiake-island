## 四、写作目标与写作目的的候选排列关系

### 4.1 一对多关系

一个写作目标可产生多个写作目的：

```
GOAL-A 评价药物价值
├── PURP-A2 赛道重定义后产品定位
├── PURP-B6 阳性结果边界解读
└── PURP-A3 获批+格局判断

GOAL-B 比较药物优劣
├── PURP-A1 多药比较评价
└── PURP-A4 机制差异驱动的同类药比较

GOAL-G 展示赛道格局
├── PURP-A2 赛道重定义后产品定位
├── PURP-C3 失败链串讲型赛道综述
└── PURP-C4 困局比喻型赛道综述

GOAL-F 解读临床结果
├── PURP-B6 阳性结果边界解读
├── PURP-B4 替代终点-OS关系解读
├── PURP-B5 不良反应-疗效关联解读
├── PURP-B7 循证缺口补足解读
└── PURP-B8 后线困局解释
```

### 4.2 多对一关系

多个写作目标可产生同一写作目的：

```
PURP-B6 阳性结果边界解读 ← GOAL-A, GOAL-F
PURP-A2 赛道重定义后产品定位 ← GOAL-A, GOAL-G
```

### 4.3 排列规则的作用

排列规则决定"写作目标 → 写作目的"的具体形态：

| 排列规则 | 作用 |
|----------|------|
| 细节买权 | "大判断必须用细节买到发言权" |
| 可迁移的是结构作用 | "不是迁移素材，而是迁移结构作用" |
