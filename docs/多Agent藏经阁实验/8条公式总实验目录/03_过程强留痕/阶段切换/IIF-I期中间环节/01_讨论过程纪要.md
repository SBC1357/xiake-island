# IIF-I期中间环节讨论过程纪要

状态：已记录  
日期：2026-04-05  
用途：简要保留 `IIF` 提前终止并转入“是否重回 I期”讨论时的关键收敛路径

## 过程摘要

1. 讨论起点是：`IIF` 的复验输入是否应正式收口为 `21` 条最小单元基底 + `14` 条组合处理映射 + `7` 条观察残留。
2. 随后讨论焦点前移到更上游的问题：旧 `动作 / 构件 / 规则 / 拓扑 / 人格` 与 `IIF` 的新裁判口径之间已经发生漂移，继续在混坐标中启动 `32` 篇复验会继续增熵。
3. 讨论进一步转向“最小单元、组合规则、组合单元”的上游定义，并在回扫既往 `藏经阁 L1/L2` 资产后，确认当前系统确实存在可写成公式的内容链和逻辑链。
4. 在逐条讨论后，当前线程把主链收成 `8` 条公式：
   - `写作目标 * 排列规则 = 写作目的`
   - `写作目的 * 人格 = 写作范围`
   - `写作范围 * 最小内容单元 = 最小有效内容单元`
   - `写作范围 * 最小逻辑单元 = 最小有效逻辑单元`
   - `最小有效内容单元 * 内容单元组合规则 = 内容组合`
   - `最小有效逻辑单元 * 逻辑组合规则 = 组合逻辑`
   - `内容组合 * 组合逻辑 = 文章大纲`
   - `写作范围 * 文章大纲 = 有效大纲`
5. 讨论同时形成新的范围切分：这 `8` 条公式只接管写作规范层；`L1` 继续承接语域、去 AI 味、表达库存等通用表达层资产；`M4` 被讨论为“通用写作规则的医学映射”，更接近 `L2.5`。
6. 最终判断出现阶段性转折：这轮讨论已经不再是“在 `II期` 旧对象框架里继续加一轮复验”，而是开始重写范围内资产的解释主轴，因此需要单独记录“是否重回 `I期`”。
