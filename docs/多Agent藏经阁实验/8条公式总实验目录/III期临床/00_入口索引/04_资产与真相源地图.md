# III期临床资产与真相源地图

状态：工作版 / 真相源地图
日期：2026-04-11
用途：告诉新 Agent 缺规则、缺样稿、缺资产、缺生产接手口径时，应该先去哪找；避免在 `III期临床` 里乱补第二份真相源

## 这张图解决什么

1. 让新 Agent 知道 `III期临床` 里有哪些正式层。
2. 让新 Agent 知道 `III期临床` 外还必须去哪找真相源。
3. 让新 Agent 知道“引用外部”不是缺陷本身，真正要避免的是没有指路、没有真相源边界。

## 先记住这条总原则

1. `III期临床` 负责收束规则、裁决和架构。
2. `多Agent藏经阁实验` 内、`III期临床` 外的小规模低频必需资产，可以冻结进 `06_内循环执行资产包`，但只允许进冻结副本，不允许复制成第二套活规则。
3. `藏经阁`、生产接入包、样稿库、runtime 过程结果各管各的，不在这里重抄第二份。
4. 缺东西先回查真相源，不要自己补新口径。

## A. III期临床 内的真相源

### 1. 00_入口索引

解决什么：

1. 冷启动怎么进门
2. 先读什么
3. 当前唯一主任务是什么
4. 外部真相源在哪

### 2. 01_散落经验裁决层

解决什么：

1. 经验从哪里来
2. 一条经验当前是保留、收紧、合并、删除还是待验证
3. 哪些经验还不能直接升成正式规则

### 3. 02_节点模块规则明细层

解决什么：

1. 节点级工作版规则
2. 通过 / 不通过
3. 回流方向
4. 标准件预留位和已落下来的工作版标准件

### 4. 03_完整架构层

解决什么：

1. 系统总图
2. 节点之间怎么接
3. 哪些内容属于系统级关系，不属于节点细则

### 5. 06_内循环执行资产包

解决什么：

1. 把 `8条公式总实验目录` 内、`III期临床` 外的小规模低频必需资产迁进本目录，先让 `III期` 对旧历史层减依赖
2. 给当前执行留一层本地冻结副本和分层清单
3. 明确哪些资产已内化、哪些仍只保留路径引用、哪些仍在 mix 待裁

使用边界：

1. 这里只放冻结副本和分层清单，不放当前阶段、当前 blocker、当前下一步
2. 这里只放小规模低频件，不放 `藏经阁`、生产接入包和 runtime 全量正文
3. 这里只作辅助执行层，不替代 `02_节点模块规则明细层` 的规则正文

## C. 常见问题先去哪找

1. 不知道当前该先做什么：
   - 先看 `00_入口索引/03_当前阶段任务卡.md`
2. 不知道先读哪几份：
   - 先看 `00_入口索引/01_冷启动阅读顺序.md`
3. 不知道某条经验能不能升层：
   - 先看 `01_散落经验裁决层`
4. 不知道当前节点规则是什么：
   - 先看 `02_节点模块规则明细层`
5. 不知道系统总图和节点关系：
   - 先看 `03_完整架构层`
6. 不知道八公式、模块卡、接口路由的旧真相源：
   - 回 `01_阶段实验方法`
7. 不知道稿型总表、冷启动配置、主样稿母本从哪来：
   - 回 `00_侠客岛现有成果接入包`
8. 不知道写作资产、医学 playbook、学科资产从哪来：
   - 回 `D:\\汇度编辑部1\\藏经阁\\l1~l4`
9. 不知道哪些多Agent内部小规模资产已经迁入 III期：
   - 先看 `06_内循环执行资产包/00_说明与分层清单.md`
