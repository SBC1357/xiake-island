# IIIA期实验方法

状态：工作版 / 阶段实验方法
日期：2026-04-10

## 方法核心

1. 先以 `03_完整架构层` 作为边界来源。
2. 先把启动执行前必须冻结的入口、输入、输出、评分和停闸条件写死。
3. 本轮不进入真实试运行和结果裁决。

## 本阶段的方法纪律

1. 先收唯一入口、唯一状态源、唯一准备清单，不平均铺开所有节点。
2. 先把旧执行记录和当前启动准备分开，再谈实跑。
3. 没有直接证据，不把“已准备”说成“已可执行”。
4. 当前结果模板全部按未激活处理。

## 本阶段的方法产物

1. 启动前准备清单
2. 单题全貌模板
3. 结果模板冻结说明
